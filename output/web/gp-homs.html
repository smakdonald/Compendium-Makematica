    Chapter 2 Homomorphisms and Isomorphisms

    Section 2.1 Group Homomorphisms

    Definition 2.1.

    If \(G\) and \(H\) are groups, a homomorphism from \(G\) is \(H\) is a function \(f: G \to H\) such that \(f(x \cdot_G y) = f(x) \cdot_H f(y)\text{,}\) where \(\cdot_G\) and \(\cdot_H\) denote the multiplication rules for \(G\) and \(H\text{,}\) respectively.

    Example 2.2.

    Prove the following
    1. The identity map is a group isomomorphism for any group \(G\text{.}\)
    2. The exponential map \(\exp: ({\mathbb{R}}, +) \to ({\mathbb{R}}\setminus \{0\}, \cdot)\) is a homomorphism. So is \(\ln: ({\mathbb{R}}_{>0}, \cdot) \to ({\mathbb{R}}, +)\text{.}\) In fact, these maps are inverse to each other so we obtain an isomorphism . \(({\mathbb{R}}, +)\cong ({\mathbb{R}}_{>0}, \cdot)\text{.}\)
    3. For any positive integer \(n\) and field \(F\text{,}\) the map determinant map \(\det: \operatorname{GL}_n(F) \to (F \setminus \{0\}, \cdot)\) is a group homomorphism.

    Lemma 2.3.

    If \(f: G \to H\) is a homomorphism of groups, then
    1. \(f(e_G) = e_H\) and
    2. \(f(x^{-1}) = f(x)^{-1}\text{.}\)

    Corollary 2.4.

    Let \(G\) be a group. Prove there exists a unique group homomorphism \(\varphi:G\to\Z\text{.}\)

    Definition 2.5.

    Let \(\varphi: G \to H\) is a homomorphism of groups. The kernel of \(\varphi\) is the set
    \begin{equation*} \ker(\varphi) := \{g\in G \mid \varphi(g)=e_H\}. \end{equation*}

    Definition 2.6.

    Let \(\varphi: G \to H\) is a homomorphism of groups. The image of \(\varphi\) is the set
    \begin{equation*} \im(\varphi) := \{\varphi(g) \mid g \in G\}. \end{equation*}

    Proposition 2.7.

    1. If \(f: G \to H\) is a homomorphism of groups, then the set-theoretic image of \(f\) is a subgroup of \(H\text{.}\)
    2. If \(f: G \to H\) is a homomorphism of groups, then the kernel of \(f\) is a subgroup of \(G\text{.}\)
    3. Prove that if \(f:G\to H\) is a group homomorphism and \(K\leq H\) then the preimage of \(K\text{,}\) defined as \(f^{-1}(K)=\{g\in G | f(g)\in K\}\text{,}\) is a subgroup of \(G\text{.}\)

    Proposition 2.8.

    1. A group homomorphism \(f: G \to H\) is injective if and only if \(\operatorname{ker}(f) = \{e_G\}\text{.}\)
    2. A group homomorphism \(f: G \to H\) is surjective if and only if \(\operatorname{im}(f) = H\text{.}\)

    Problem 2.9.

    1. Find, with justification, an injective group homomorphism \(f:S_n \to \GL_n(\R)\text{.}\)
    2. Find, with justification, an injective group homomorphism \(g:D_{2n}\to S_n\text{.}\)

    Section 2.2 Group Isomorphisms

    Definition 2.10.

    A homomorphism \(f: G \to H\) is called an isomorphism if there exists a homomorphism \(g: H \to G\) such that \(f \circ g = \text{id}_H\) and \(g \circ f = \text{id}_G\text{.}\)
    If \(f:G\to H\) is an isomorphism, \(G\) and \(H\) are called isomorphic, written \(G\cong H\text{.}\)

    Remark 2.11.

    I have defined the notion of isomorphism in Definition Definition 2.10 differently than given in the textbook. The reason is that the correct meaning of the word “isomorphism” in any context (sets, groups, rings, fields, topological spaces, whatever) is always “a morphism that has a two-sided inverse”. In many contexts, such as sets, groups, rings and fields this turns out to be equivalent to the notion of being “one-to-one and onto”. But there are contexts in which this is not the case. For example a one-to-one and onto continuous map of topological spaces need not be a homeomorphism. (A homeomorphism is a term one uses for isomorphism of topological spaces, for historical reasons.)

    Proposition 2.12.

    Suppose \(f: G \to H\) is a group homomorphism. Then \(f\) an isomorphism if and only if \(f\) is bijective (one-to-one and onto).

    Definition 2.13.

    An isomorphism invariant is a property \(P\) such that whenever \(G \cong H\) and \(G\) has \(P\) then \(H\) has \(P\text{.}\)

    Theorem 2.14.

    The following are isomorphism invariants:
    1. the order of the group,
    2. the set of orders of elements in the group,
    3. being abelian,
    4. the order of the center of the group,
    5. being finitely generated,

    Problem 2.15.

    Decide whether the groups \(Q_8\) and \(D_8\) are isomorphic or not, with justification.

    Problem 2.16.

    1. \(S_n\cong S_m\) if and only if \(n=m\text{.}\)
    2. \(\displaystyle \mathbb{Z}/6\ncong S_3\)
    3. \(\displaystyle D_{24}\ncong S_4\)

    Definition 2.17.

    Let \(G\) be a group. An isomorphism \(\phi:G\to G\) is called an automorphism of \(G\text{.}\)

    Problem 2.18.

    Prove that the set of automorphisms of \(G\text{,}\) denoted \(\operatorname{Aut}(G)\text{,}\) is a group under composition.

    Problem 2.19.

    Let \(G\) be a group. Prove that the map \(f:G\to G\) given by \(f(a)=a^{-1}\) for all \(a\in G\) is an automorphism if and only if \(G\) is abelian.

    Problem 2.20.

    Let \(G\) be a group and \(x\in G\text{.}\) Define a map \(\psi_x:G\to G\) by \(\psi_x(a)=xax^{-1}\) for all \(a\in G\text{.}\)
    1. Prove that \(\psi_x\in \operatorname{Aut}(G)\) for all \(x\in G\text{.}\)
    2. Prove that \(\{\psi_x\mid x\in G\}\) is a subgroup of \(\operatorname{Aut}(G)\text{.}\)

    Problem 2.21.

    Prove that if groups \(G\) and \(H\) are isomorphic then:
    1. \(Z(G)\cong Z(H)\) and
    2. \(\Aut(G)\cong \Aut(H)\text{.}\)

    Problem 2.22.

    Find an example of groups \(G,H\) such that there is an isomorphism \(G\times H\cong G\) but \(H\) is not trivial. Note: since \(G\times H\cong G\) can be rewritten as \(G\times H\cong G\times \{e\}\text{,}\) the above shows that in general one cannot cancel groups in isomorphisms between direct products.

    Section 2.3 Cyclic Groups: Revisited

    Problem 2.23.

    Prove that if \(m,n\) are positive integers such that \(\gcd(m,n)=1\) then there is a group isomorphism \(C_{mn}=C_m\times C_n\text{.}\)

    Theorem 2.24.

    Let \(G=\langle x\rangle\text{,}\) where \(x\) has finite order \(n\text{.}\) Then
    1. \(G=\{e,x,\ldots,x^{n-1}\}\text{.}\)
    2. There is a bijection
      \begin{equation*} \Psi : \{\text{divisors of } |G|\} \to \{\text{subgroups of } G\} \text{ given by } \Psi(d) = \langle x^{\frac{|G|}{d}} \rangle \end{equation*}
      for each divisor \(d\) of \(|G|\text{.}\) Moreover, for each subgroup \(H\) of \(G\text{,}\) \(\Psi^{-1}(H) = |H|\text{.}\) In particular, all subgroups of \(G\) are cyclic and there is a unique subgroup of each order.

    Proposition 2.25. Universal Mapping Property (UMP) for Cyclic Groups.

    Assume \(G = \langle x \rangle\) and let \(H\) be any group. If \(|x| = n < \infty\text{,}\) then for each \(y \in H\) such that \(y^n = e\text{,}\) there is a unique group homomorphism
    \begin{equation*} f: G \to H \end{equation*}
    such that \(f(x) = y\text{.}\) If \(|x| = \infty\text{,}\) then for each \(y \in H\text{,}\) there is a unique group homomorphism
    \begin{equation*} f: G \to H \end{equation*}
    such that \(f(x) = y\text{.}\) In both cases this unique group homomorphism is given by \(f(x^i)=y^i\) for any \(i\text{.}\)

    Remark 2.26.

    This is a particular case of the universal mapping property of a presentation (which we will covered later), since a cyclic group is either presented by \(\langle x | x^n = e \rangle\) or \(\langle x \mid -\rangle\text{.}\)

    Theorem 2.27. Classification Theorem for Cyclic Groups.

    Every infinite cyclic group is isomorphic to \(C_\infty\text{.}\) Every cyclic group of order \(n\) is isomorphic to \(C_n\text{.}\)
