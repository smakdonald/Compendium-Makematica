    Section 1.1 First Definitions

    Subsection 1.1.1 Groups

    Definition 1.1.1. Binary Operation.

    A binary operation on a set \(S\) is a function
    \begin{equation*} -\cdot-:S\times S\to S,\text{given by } (x,y)\to x\cdot y. \end{equation*}

    Remark 1.1.2.

    We often write \(xy\) instead of \(x\cdot y\) for simplicity.

    Definition 1.1.3.

    A group is a pair \((G, \cdot)\) where \(G\) is a set and \(\cdot\) is a Binary Operation on \(G\text{,}\) called group multiplication, satisfying the following properties:
    1. (Associativity) for all \(x,y,z\in G\) we have \((x\cdot y)\cdot z=x\cdot (y\cdot z)\)
    2. (Identity Element) there exists \(e\in G\) such that \(e·x=x·e=x\) for all \(x\in G\)
    3. (Inverses) for each \(x\in G\text{,}\) there is an element \(y\in G\) such that \(xy=e=yx\text{.}\)

    Remark 1.1.4.

    Although a group is a pair, we will usually refer to the group by only naming the underlying set, \(G\text{.}\)

    Remark 1.1.5.

    We say that “\(S\) is closed under the operation \(\cdot\)”, when we want to emphasize that for any \(x,y\in S\) the result of the operation, \(xy\text{,}\) is an element of \(S\text{.}\) However note that closure is really part of the definition of a binary operation on a set, and it is implicitly assumed whenever we consider such an operation.

    Definition 1.1.6.

    A group \(G\) is an abelian group if \(\cdot\) is commutative; i.e., \(x \cdot y = y \cdot x\) for all \(x,y \in G\text{.}\) Often, but not always, the group operation for an abelian group is written as \(+\) instead of \(\cdot\text{.}\) In this case the inverse of an element \(x\) is written \(-x\text{.}\)

    Example 1.1.7. Group Examples.

    Prove each of the following:
    1. The trivial subgroup is the group with a single element \(\{e\}\)
    2. \((\mathbb{Z}, +)\text{,}\) \((\mathbb{Q}, +)\text{,}\) \(({\mathbb{R}}, +)\) and \((\mathbb{C},+)\) are abelian groups.
    3. For any set \(S\text{,}\) the permutations on \(S\)
      \begin{equation*} \rm{Perm}(S)=\{f:S\to S \ | \ f \text{ is a bijection}\} \end{equation*}
      form a non-abelian group under composition.
    4. For any field \(F\) and positive integer \(n\text{,}\) let
      \begin{equation*} \operatorname{GL}_n(F)=\{ \text{invertible } n \times n \text{ matrices with entries in } F\}. \end{equation*}
      By invertible I mean those matrices that have two-sided inverses, but in turns out that if an \(n \times n\) matrix has a left inverse then it it is automatically a right inverse too, and vice versa. Then \(\operatorname{GL}_n(F)\) is a non-abelian group under matrix multiplication.
    5. For any \(n\text{,}\) let \(\mathbb{Z}/(n)\) denote the integers modulo \(n\text{.}\) Then \((\mathbb{Z}/(n), +)\) is an abelian group where \(+\) denotes addition modulo \(n\text{.}\)
    6. For any field \(F\) (e.g., \(\mathbb{Q}\text{,}\) \({\mathbb{R}}\text{,}\) \(\mathbb{C}\) or \(\mathbb{Z}/p\) for a prime \(p\)), the set \(F^\times := F \setminus \{0\}\) is an abelian group under the usual multiplication.
    Solution.
    Note that \(GL_1(F)\) is the same thing as \((F^\times, \cdot)\text{.}\)

    Problem 1.1.8.

    These example are infinite if \(F\) is infinite, but if we take \(F = \mathbb{Z}/p\) for a prime \(p\text{,}\) then \(\operatorname{GL}_n(\mathbb{Z}/p)\) is a finite group. Can you find its cardinality?

    Problem 1.1.9.

    Give an example of a pair \((G,\cdot)\) that satisfies axioms 1 and 2 of Definition 1.1.3 and an element of \(G\) that has a left inverse but not a right inverse.
    Solution.

    Proposition 1.1.10.

    In a group \(G\text{,}\) the element \(e\) satisfying the second axiom of Definition 1.1.3 is unique, and we thus refer to it as the identity element of \(G\text{.}\)

    Proof.

    If \(ex = x = xe\) and \(e'x = x = xe'\) for all \(x\text{,}\) then \(e = ee' = e'\text{.}\)

    Proposition 1.1.11.

    In a group \(G\text{,}\) for each \(x\text{,}\) the element \(y\) satisfying the last axiom of Definition 1.1.3 is unique.

    Proof.

    Proof. For a given \(x\text{,}\) if \(yx = xy = e\) and \(zx = xz = e\) fo some \(y\) and \(z\text{,}\) then \(z = ez = (yx)z = y(xz) = ye = y\text{.}\)

    Remark 1.1.12.

    We will call the element \(y\) satisfying the last axiom of Definition 1.1.3 the inverse of \(x\) and we will henceforth denote it by \(x^{-1}\text{.}\)

    Lemma 1.1.13. Properties of Groups.

    If \(G\) is a group and \(x,y,z, a_1,\ldots,a_n \in G\text{,}\) then:
    1. if \(xy = xz\text{,}\) then \(y=z\text{.}\)
    2. if \(yx = zx\text{,}\) then \(y = z\text{.}\)
    3. \((x^{-1})^{-1} = x\text{.}\)
    4. \((a_1 \dots a_n)^{-1} = a_n^{-1} \dots a_1^{-1}\text{.}\)
    5. \((x^{-1}yx)^n = x^{-1}y^nx\text{.}\)

    Subsection 1.1.2 Order

    Definition 1.1.14.

    In a group \(G\text{,}\) the order of an element \(x\) is the least positive integer \(n\) such that \(x^n = e\text{.}\) If no such \(n\) exists, we say \(x\) has infinite order. We write \(|x|\) for the order of \(x\text{.}\)

    Definition 1.1.15.

    The order of a group \(G\) is the cardinality of the set \(G\text{,}\) denoted \(|G|\text{.}\)

    Problem 1.1.16.

    Prove that any group of order \(4\) is abelian.
    Solution.

    Proposition 1.1.17.

    Let \(G\) be a group and \(x \in G\) any element.
    1. Prove that if \(|x| = n\text{,}\) then \(e, x, \dots, x^{n-1}\) are all distinct elements of \(G\text{.}\)
    2. Prove that if \(|x| = \infty\text{,}\) then \(x^i \ne x^j\) for all integers (positive or negative) \(,j\) with \(i \ne j\text{.}\)
    3. Conclude \(|x| \leq |G|\) in all cases.

    Problem 1.1.18.

    Let \(G, H\) be groups and let \(x, y\) be elements of \(G\text{.}\)
    1. Prove that \(|x|=\left|\{x^n \ | n\in \Z \}\right|\) and in particular \(|x| \leq |G|\text{.}\)
    2. Prove or disprove: if \(x,y\) have finite order then \(xy\) has finite order.

    Problem 1.1.19.

    Let \(G\) be a group.
    1. Let \(g \in G\) be an element of finite order. Show that \(g^m\) has finite order for any integer \(m \geq 0\text{,}\) and in fact
      \begin{equation*} |g^m| = \frac{\lcm(m,|g|)}{m} = \frac{|g|}{\gcd(m, |g|))}. \end{equation*}
    2. Prove that for all \(g, h\) in \(G\text{,}\) \(|gh| = |hg|\) holds.

    Lemma 1.1.20.

    If \(x^m = e\) then \(|x| \mid m\text{.}\)
