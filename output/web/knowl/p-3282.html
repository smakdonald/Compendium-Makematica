If \(G\) is a group, \(A \leq G\text{,}\) \(C \leq G\text{,}\) \(B \unlhd A\text{,}\) and \(D \unlhd C\text{,}\) then \((A \cap C)B \leq G\text{,}\) \((A \cap C)D \leq G\text{,}\) \((A \cap D)B \unlhd(A \cap C)B\text{,}\) \((B \cap C)D \unlhd(A \cap C)D\) and there is an isomorphism
\begin{equation*} (A \cap C)B /(A \cap D)B \cong (A \cap C)D/(B \cap C)D. \end{equation*}
