Solution 4.3.4.1.
Suppose \(u,v \in X\) lie in the same orbit. Then \(u = gv\) for some \(g \in G\text{.}\) Then \(a \in G_u\) if and only \(au = u\text{,}\) which is if and only if \(agv = gv\text{,}\) which is if and only if \((g\inv ag)v = v\text{,}\) which is if and only \(if g\inv ag \in Gv\text{,}\) which is if and only if \(a \in gGvg\inv.\)
Thus, \(G_u = gG_vg\inv\text{.}\) Now, if \(G\) acts transitively on \(X\text{,}\) then \(X = \{gv |g \in G\}\) for any \(v \in X\text{.}\) Hence, by the argument above, for any \(u \in X, G_u = gG_vg\inv\) for some \(g \in G\text{.}\)
Since the kernel of the action is the intersection of all the stabilizers, we obtain the desired conclusion.
