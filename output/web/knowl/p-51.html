  • A magma (or binary structure) is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\text{.}\)
  • A semigroup is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\) satisfying the associative property.
  • A monoid is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\) satisfying the associative and identity properties.
