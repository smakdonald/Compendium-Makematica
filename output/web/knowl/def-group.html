Definition 1.10. Group.

A group is a Monoid \((G,\cdot)\) with the following additional property:
  • Inverse Axiom.
    for each \(x\in G\text{,}\) there is an element \(y\in G\) such that \(xy=e=yx\text{.}\)
When the operation of a group is \(\cdot\text{,}\) we refer to \(G\) as a multiplicative group. If the operation \(+\) is used, then \(G\) is referred to as an additive group.
