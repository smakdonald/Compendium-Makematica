Definition 1.6. Group.

  1. A group is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\) satisfying the associative, identity, and inverse properties.  2 
  2. An abelian group is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\) satisfying the associative, identity, inverse, and commutative properties.
