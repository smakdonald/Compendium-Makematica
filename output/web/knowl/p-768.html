Suppose \(G\) acts on a set \(X\) and let \(u,v\in X\text{.}\) Suppose \(u\) and \(v\) lie in the same orbit. Prove that \(G_u=gG_vg^{-1}\) for some \(g\in G\text{.}\) Conclude that if the action is transitive (i.e., there is precisely one orbit) then for any \(u\in X\) the kernel of the action is
\begin{equation*} \bigcap_{g\in G} gG_ug^{-1}. \end{equation*}
