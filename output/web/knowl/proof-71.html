Proof

Suppose \(R\) is an integral domain and that \(r\) is prime. Then \(r \ne 0\) and \(r\) is not a unit. Suppose \(r = yz\text{.}\) Then \(yz \in (r)\) and hence by definition either \(y \in (r)\) or \(z \in (r)\text{.}\) If \(y \in (r)\text{,}\) we have \(y = rt\) for some \(t\) and so \(y = yzt\text{.}\) Since \(r \ne 0\text{,}\) \(y \ne 0\text{,}\) and \(R\) is an integral domain, we must have \(zt = 1\text{,}\) showing that \(z\) is a unit.
Assume \(R\) is a PID and that \(r\) is irreducible. Since \(r\) is not a unit, \((r)\) is a proper ideal and hence is contained in a maximal ideal \(M\) by Theorem . We show \((r)= M\) and hence \((r)\) is prime. Since \(R\) is a PID, \(M = (y)\) for some \(y\text{.}\) So \(x = yt\) for some \(t\text{.}\) But \(x\) is irreducible and \(y\) is not a unit, which forces \(t\) to be a unit and hence \((x) = (y) = M\text{.}\)
