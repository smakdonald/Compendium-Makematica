A Euclidean domain (ED) is a domain \(R\) together with a function \(N: R \to \Z_{\geq 0}\) such that \(N(0) = 0\) and the following property holds: for any two elements \(a, b \in R\) with \(b \ne 0\text{,}\) there are elements \(q\) and \(r\) of \(R\) such that
\begin{equation*} a = qb + r \;\text{and either}\; r = 0 \;\text{or}\; N(r) < N(b). \end{equation*}
