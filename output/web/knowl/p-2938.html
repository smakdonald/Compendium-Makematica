So instead let's try a different approach. Let \(L = \Q(\sqrt[4]{5})\text{.}\) Then since \(x^4 - 5\) is irreducible in \(\Q[x]\) by Eisenstein, we have \([L: \Q] = 4\text{.}\) Since \(F = L(i)\) and \(i\) is a root of \(x^2 +1\text{,}\) we have \([F:L] \leq 2\text{.}\) But \(F \ne L\) since \(F \subseteq \R\text{.}\) Note that \([F:L] = 1\) if and only if \(F = L\text{.}\) Thus \([F:L] = 2\text{.}\)
