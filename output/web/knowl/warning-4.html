Warning B.10.

Beware! The order relation \(\unlhd\) is not transitive. For example, you showed that \(V \unlhd S_4\) and, since \(V\) is abelian, the subgroup \(M = \{e,(12)(34)\}\) is normal in \(V\text{.}\) But \(M\) is not normal in \(S_4\text{,}\) since for example
\begin{equation*} (1 3) [(1 2)(3 4)] (1 3)^{-1} = (3 2)(1 4) \notin M. \end{equation*}
So in the above definition it is not necessarily true that each \(N_i\) is normal in \(G\text{.}\)
