Proof

It is clear that this map is additive, as each \(\iota_i\) and \(\pi_j\) is. Observe that \(\pi_j \iota_i\) is the identity on \(M\) if \(i=j\text{,}\) and the zero map otherwise and that \(1_{M^{\oplus n}} = \sum_{k} \iota_k \pi_k\text{.}\)
The map
\begin{equation*} \xymatrix@R=1mm{ \operatorname{End}_R(M^{\oplus n}) &\ar[l]_-{\zeta} \operatorname{Mat}_n(\operatorname{End}_R(M)) \\ \sum_{i,j} \iota_i \alpha_{i,j} \pi_j & \ar@{|->}[l] [\alpha_{i,j}]_{i,j} } \end{equation*}
is a two-sided inverse for \(\Theta\text{:}\)
\begin{equation*} \zeta (\Theta(\phi)) = \zeta ( [\pi_i \phi \iota_j]_{i,j} ) = \sum_{i,j} \iota_i \pi_i \phi \iota_j \pi_j =(\sum_i \iota_i \pi_i) \phi (\sum_j \iota_j \pi_j)=\phi, \quad\text{and} \end{equation*}
\begin{equation*} \Theta(\zeta([\alpha_{i,j}]_{i,j}]))_{k,\ell} = \Theta ( \sum_{i,j} \iota_i \alpha_{i,j} \pi_j)_{k,\ell} = \pi_k (\sum_{i,j} \iota_i \alpha_{i,j} \pi_j) \iota_\ell = \sum_{i,j} (\pi_k \iota_i) \alpha_{i,j} (\pi_j \iota_\ell) = \alpha_{k,\ell}. \end{equation*}
To see that \(\Theta\) respects multiplication, we have
\begin{equation*} [\Theta (\psi) \Theta(\phi) ]_{i,j}= \sum_k (\pi_i \psi \iota_k) (\pi_k \phi \iota_j) = \pi_i \psi\phi \iota_j = \Theta(\psi \phi)_{i,j}.\qedhere \end{equation*}
