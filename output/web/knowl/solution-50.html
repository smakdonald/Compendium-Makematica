Solution 3.1.

Let \(G\) be a group that acts on a set \(A\text{,}\) and \(H\) a subgroup of \(G\) such that for any \(a, b \in A\) there exists a unique \(h \in H\) with \(ha = b\text{.}\)
Suppose there exists some \(h\in H\) such that \(h\in G_a\text{.}\) Then \(ha=a\text{.}\) But \(ea=a\) by the definition of group action. As the \(h\in G_a\) is unique, we see \(h=e\text{.}\) Thus \(H\cap G_a=\{e\}\text{.}\)
Let \(g\in G\) and \(a\in A\text{.}\) If \(g\in H\) then we can write \(g=ge\text{,}\) as \(e\in G_a\text{.}\) Suppose \(g\not\in H\) and \(ga=b\) for some \(b\in A\text{.}\) There exists an \(h\in H\) such that \(ha=b\text{.}\) Then \(ha=ga\text{,}\) so \(h\inv g a=a\text{,}\) so \(h\inv g\in G_a\text{.}\) Thankfully, we can know write \(g=h(h\inv g\)), and so \(G=HG_a\text{.}\)
If \(H\subseteq Z(G)\) then we have \(HG_a=G_aH\text{,}\) which means \(HG_a\leq G\text{.}\) From Part (a) we have a trivial intersection, making \(G\) the internal direct product of \(H\) and \(G_a\text{.}\)
