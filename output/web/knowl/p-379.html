  1. If \(x\in G\) and \(G\) is a finite group, then \(|x|\) divides \(|G|\text{.}\)
  2. If \(G\) is a finite group, then \([G:H]=\frac{|G|}{|H|}\)
  3. If \(|G|=n\) then \(g^n=e\) for all \(g\in G\)
  4. (Groups of prime order are cyclic) If \(|G|=p\) for some prime \(p\text{,}\) then \(G\) is cyclic and any non-identity element of \(G\) generates \(G\text{.}\)
