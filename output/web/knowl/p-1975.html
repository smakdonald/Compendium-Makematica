The following are equivalent for a field \(L\text{:}\)
  1. \(L\) is algebraically closed.
  2. Every non-constant polynomial with coefficients in \(L\) splits completely into (not necessarily distinct) linear factors.
  3. There are no non-trivial algebraic extensions of \(L\text{:}\) If \(L \subseteq E\) is an algebraic field extension then \(L = E\text{.}\)
