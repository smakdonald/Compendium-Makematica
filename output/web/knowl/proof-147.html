Proof

Assume that \(M\) has finite length. Suppose that \(M\) is not Noetherian. Then there is a chain
\begin{equation*} M_0 \subsetneqq M_1 \subsetneqq M_2 \subsetneqq \cdots \end{equation*}
Since each \(M_i\) is a submodule of \(M\text{,}\) its length is finite, and is a nonnegative integer. Then \(\ell(M_0) < \ell(M_1) < \ell(M_2) < \cdots \leq \ell(M)\text{,}\) which yields a contradiction. The argument that \(M\) is Artinian is similar.
Now assume that \(M\) is both Noetherian and Artinian. We will construct a composition series for \(M\text{.}\) We can assume that \(M\neq 0\text{.}\) Consider the collection of proper submodules of \(M\text{.}\) This is nonempty, so has a maximal element \(M^1\) by the Noetherian hypothesis. We must have \(M/M^1\) is simple, or else there is a module in between \(M^1\) and \(M\text{.}\) Using Noetherianity again, if \(M^1\neq 0\) (weâ€™re done otherwise), there is a maximal proper submodule of \(M^1\text{;}\) call it \(M^2\text{.}\) This process yields a descending chain with simple quotients, and this must stop (i.e., yield \(M^i=0\) for some \(i\)) by the Artinian hypothesis. Thus, there is a composition series for \(M\text{.}\)
