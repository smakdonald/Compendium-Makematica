Conversely, suppose \(\alpha = \alpha_1 \cdots \alpha_k\) and \(\beta = \beta_1 \cdots \beta_k\) are decompositions into disjoint cycles and that \(\alpha_i, \beta_i\) both have length \(p_i \geq 2\) for all \(i\text{.}\) We need to prove \(\alpha\) and \(\beta\) are conjugate. Let’s start with the case \(k = 1\text{:}\) given two cycles of the same length
\begin{equation*} \alpha = (i_1 \, \dots \, i_p) \text{ and } \beta = (j_1 \, \dots \, j_p). \end{equation*}
If \(\sigma\) is any permutation such that \(\sigma(i_m) = j_m\) for all \(m = 1, \dots, p\text{,}\) then \(\sigma\alpha\sigma^{-1} = \beta\) by Lemma B.2.
