Example 1.15.

  1. \(\displaystyle |e|=1\)
  2. \(|1|=n\) in \(\Z/(n)\)
  3. In the additive groups \(\Z\text{,}\) \(\Q\text{,}\) \(\R\text{,}\) and \(\C\) every nonzero (i.e., nonidentity) element has infinite order.
  4. In the additve group \(\Z/(6)\text{,}\) the element \(2\) has order \(3\) because \(2+2=4\) and \(2+2+2=0\text{.}\)
