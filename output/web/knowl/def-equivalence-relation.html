Definition A.27.

Anequivalence relation\sim on a set X is a subset S of X \times X (where (a,b) \in S is written a \sim b) that satisfies the following for all a,b,c in X: (1) a \sim a (reflexive), (2) a \sim b implies b \sim a (symmetric), and (3) a \sim b and b \sim c implies a \sim c (transitive).
Theequivalence classof an element a of X is[a]:= \{b | b \sim a\}.
The notationX/\simdenotes theset of equivalence classes, also called thequotientof X with respect to \sim .
The function q:X \to X/\sim defined by q(p) := [p] for all p \in X is called theequivalence map
