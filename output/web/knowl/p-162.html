Prove that there are group isomorphisms between the following pairs of groups:
  1. \(\left(\Q/\Z, +\right)\cong \left(\mu,\cdot\right)\text{,}\) where \(\mu=\{z\in \C \mid z^n=1 \text{ for some integer } n\}\text{.}\)
  2. \((\C^\times,\cdot)/\mu_m\cong (\C^\times,\cdot)\) for any integer \(m\geq 1\text{,}\) where \(\mu_m=\{z\in \C \mid z^m=1\}\text{.}\)
