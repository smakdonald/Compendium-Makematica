Finally, we need to prove we've found all groups of order \(12\text{.}\) If \(\# G = 12\text{,}\) then \(\operatorname{Syl}_2(G) \in \{1, 3\}\) and \(\operatorname{Syl}_3(G) \in \{1, 4\}\text{.}\) \(\operatorname{Syl}_2(G) = 3\) and \(\operatorname{Syl}_3(G) = 4\) cannot both occur since that would lead to at least \(8 + 6\) elements. This gives the existence of \(N \unlhd G\) and \(H \leq G\) with either \(\# N = 4, \# H = 3\) or \(\# N = 3, \# H = 4\text{.}\) The usual reasoning gives \(H \cap N = \{e\}\) and \(HN = G\text{,}\) so that \(G \cong N \rtimes_\rho H\) for some \(\rho: H \to \operatorname{Aut}(N)\text{.}\)
