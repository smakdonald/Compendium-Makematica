Proof

If \((x) = (y)\) then \(x \in (y)\) and so \(x = yu\) for some \(u\text{.}\) Similarly \(y = xs\) and hence \(y = yus\text{.}\) Since \(R\) is a domain, either \(y = 0\) or \(su = 1\text{.}\) If \(y= 0\text{,}\) then \(x = 0 = 1 y\) and otherwise \(u\) is a unit.
If \(x = uy\) for a unit \(u\text{,}\) then \(y = u^{-1}x\) and so \(x \in (y)\) and \(y \in (x)\text{,}\) from which is follows that \((x) \subseteq (y)\) and \((y) \subseteq (x)\text{.}\)
