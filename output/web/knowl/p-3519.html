Finally let \(n\) and \(\tau\) be arbitrary. Then \(\tau\) is a product of transpositions â€” say \(\tau = \tau_r \cdots \tau_1\) with each \(\tau_i\) a transposition. Proceed by induction on \(r\text{.}\) The previous case establishes the result when \(r = 1\text{.}\) If \(r \geq 2\text{,}\) then using the previous case gives
\begin{equation*} \psi(v_\tau(1), \dots, v_\tau(n)) = -\psi(v_\tau'(1), \dots, v_\tau'(n)) \end{equation*}
where \(\tau' = \tau_{r-1} \cdots \tau_1\) and by induction \(\psi(v_\tau'(1), \dots, v_\tau'(n)) = (-1)^{r-1}\psi(v_1, \dots, v_n)\text{.}\) Hence
\begin{equation*} \psi(v_\tau(1), \dots, v_\tau(n)) = (-1)^{r}\psi(v_1, \dots, v_n) \end{equation*}
which proves the result.
