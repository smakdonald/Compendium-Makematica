To show \(B'\) is linearly indepedent, suppose
\begin{equation*} cw + \sum_{i=1}^p c_i u_i= 0 \end{equation*}
for some \(c, c_i \in F\) and some \(u_i \in (B \setminus \{v_1\})\text{.}\) If \(c \ne 0\text{,}\) then we have
\begin{equation*} w = \sum_i d_i u_i \end{equation*}
for some \(d_i\text{.}\) But this contradicts the uniqueness of \(w = a_1 v_1 + \cdots + a_n v_n\) (for note that \(u_i \ne v_1\) for all \(i\)). So we must have \(c = 0\text{.}\) But then we must also have \(c_i = 0\) for all \(i\) since \(B\) is linearly independent. This proves \(B'\) is linearly independent. This completes the proof of the sublemma.
