Theorem A.9.

Let a: G \to H, b: H \to K, and c: K \to L be functions. Then:
  1. If a and b are one-to-one then ba is one-to-one.
  2. If ba is one-to-one then a is one-to-one.
  3. If a and b are onto then ba is onto.
  4. If ba is onto then b is onto.
  5. A is a bijection if and only if a is invertible.
