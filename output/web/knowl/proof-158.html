Proof

Note that from the previous homework, we know that \([E:\Q]=8\text{,}\) and that \(E=\Q(\sqrt[4]{20},i)\text{.}\) Additionally, we know the following: - \(\a_1 = e^{\pi i/4}\sqrt[4]{5} = \cos(\frac{\pi}{4})+i\sin(\frac{\pi}{4})\sqrt[4]{5} = \frac{\sqrt{2}}{2}\sqrt[4]{5}+i\frac{\sqrt{2}}{2}\sqrt[4]{5}=\sqrt[4]{5/4}(1+i)\text{,}\) - \(\a_2 = e^{3\pi i/4}\sqrt[4]{5} = \cos(\frac{3\pi}{4})+i\sin(\frac{3\pi}{4})\sqrt[4]{5} = \frac{-\sqrt{2}}{2}\sqrt[4]{5}-i\frac{\sqrt{2}}{2}\sqrt[4]{5}=\sqrt[4]{5/4}(-1+i)\text{,}\) - \(\a_3 = e^{5\pi i/4}\sqrt[4]{5} = \cos(\frac{5\pi}{4})+i\sin(\frac{5\pi}{4})\sqrt[4]{5} = \frac{\sqrt{2}}{2}\sqrt[4]{5}+i\frac{\sqrt{2}}{2}\sqrt[4]{5}=\sqrt[4]{5/4}(-1-i)\text{,}\) and - \(\a_4 = e^{7\pi i/4}\sqrt[4]{5} = \cos(\frac{7\pi}{4})+i\sin(\frac{7\pi}{4})\sqrt[4]{5} = \frac{\sqrt{2}}{2}\sqrt[4]{5}-i\frac{\sqrt{2}}{2}\sqrt[4]{5}=\sqrt[4]{5/4}(1-i)\text{.}\) Note that by Eisenstein's Criterion using \(p=5\) we see that \(x^4+5\) is irreducible in \(\Q\text{.}\) As \(\Q\) has characteristic 0, we see that \(x^4+5\) is separable as well.
As \(E\) is the splitting field of an irreducible polynomial in \(F[x]\text{,}\) by Theorem 2.90 we see that \(|\Aut(E/\Q)|=[E:\Q]=8\text{.}\) In the Gaussian integers, it is known that if both \(a\) and \(b\) are nonzero then, \(a+bi\) is a Gaussian prime if and only if \(a^2+b^2\) is an ordinary prime. Note that \(5=(2-i)(2+i)\text{,}\) and that \(2^2+i^2=3\text{,}\) which is prime. Thus \(2+i\) is prime in \(\Z[i]\text{.}\) Thus we may invoke Eisenstein's Criterion using \(p=2+i\) to see that \(x^4+5\) is irreducible in \(\Q(i)\text{,}\) as \((2+i)^2=3-2i\) does not divide 5, but we'll prove it just to be sure.
Suppose \((3+2i)|5\text{.}\) Then \((3+2i)(a+bi)=5\text{,}\) so \(3a+3bi+2ai-2b=5\text{,}\) and thus \(3a-2b+(2a+3b)i=5\text{.}\) As 5 has no imaginary component, it must be the case that \(2a+3b=0\text{,}\) and thus that \(a=-\frac{3}{2}b\text{.}\) Substituting in, we see that \(-\frac{9}{2}b-3b=5\text{,}\) and thus that \(b=-\frac{10}{3}\text{,}\) which has historically not been an integer.
Anyway, we may thus apply Porism 2.61 to see that there exists some \(\tau\in\Aut(E/\Q(i))\) such that \(\tau(\alpha_1)=\alpha_2\text{.}\) We may regard \(\tau\) as an element of \(\Aut(E/\Q)\) since, by definition, \(\Aut(E/\Q(i))\) is a subgroup of \(\Aut(E/\Q)\text{.}\)
As \(\tau(\alpha_1)=\alpha_2\text{,}\) observe the following glory:
\begin{equation*} \begin{align} \tau(\alpha_1)&=\alpha_2\\ \tau(\sqrt[4]{5/4}(1+i))&=\sqrt[4]{5/4}(-1+i)\\ \tau(\sqrt[4]{5/4})\tau(1+i)&=\sqrt[4]{5/4}(-1+i)\\ \tau(\sqrt[4]{5/4})(1+i)&=\sqrt[4]{5/4}(-1+i)\\ \tau(\sqrt[4]{5/4})&=\frac{\sqrt[4]{5/4}(-1+i)}{(1+i)}\\ \tau(\sqrt[4]{5/4})&=\frac{\sqrt[4]{5/4}(-1+i)}{(1+i)}\frac{(1-i)}{(1-i)}\\ \tau(\sqrt[4]{5/4})&=\frac{\sqrt[4]{5/4}(2i)}{2}\\ \tau(\sqrt[4]{5/4})&=\sqrt[4]{5/4}i \end{align} \end{equation*}
With that in mind, we examine
\begin{equation*} \begin{align} \tau(\alpha_2)&=\tau(\sqrt[4]{5/4}(-1+i))\\ &=\tau(\sqrt[4]{5/4})\tau(-1+i)\\ &=\tau(\sqrt[4]{5/4})(-1+i)\\ &=\sqrt[4]{5/4}i(-1+i)\\ &=\sqrt[4]{5/4}(-1-i)\\ &=\alpha_3, \end{align} \end{equation*}
and
\begin{equation*} \begin{align} \tau(\alpha_3)&=\tau(\sqrt[4]{5/4}(-1-i))\\ &=\tau(\sqrt[4]{5/4})\tau(-1-i)\\ &=\tau(\sqrt[4]{5/4})(-1-i)\\ &=\sqrt[4]{5/4}i(-1-i)\\ &=\sqrt[4]{5/4}(1-i)\\ &=\alpha_4, \end{align} \end{equation*}
and don't forget
\begin{equation*} \begin{align} \tau(\alpha_4)&=\tau(\sqrt[4]{5/4}(1-i))\\ &=\tau(\sqrt[4]{5/4})\tau(1-i)\\ &=\tau(\sqrt[4]{5/4})(1-i)\\ &=\sqrt[4]{5/4}i(1-i)\\ &=\sqrt[4]{5/4}(1+i)\\ &=\alpha_1, \end{align} \end{equation*}
So \(\tau\) corresponds to the permutation (1 2 3 4). We have proven that \(\Aut(E/\Q(i))\) is isomorphic to a subgroup of \(S_4\) having order at most 8 and that it contains an element or order 2 and (1 2 3 4), an element of order 4. Since the subgroup generated by these two elements has order 8, we see that \(\Aut(E/\Q) \cong D_8\text{.}\) (Also by Sylow's Theorem, \(S_4\) only has one subgroup of order 8, and it is \(D_8\)).
