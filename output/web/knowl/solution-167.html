Solution G.10.1.

  1. Let \(G=S_3\text{,}\) \(H=\langle(12)\rangle\text{,}\) and \(K=\langle(13)\rangle\text{.}\) Notice that \(H\cap K=\{e\}\text{,}\) and thus \(|HK|=4\text{,}\) which does not divide \(6\text{.}\) Thus \(HK\) is not a subgroup of \(G\text{,}\) as this would contradict Lagrange's Theorem
  2. Suppose there exists some \(k\in K\) such that \(k\not\in H\text{.}\) We examine \(kH\in G/H\text{.}\) Notice that since \(G/H\) has prime order it is cyclic by [provisional cross-reference: cite], and thus generated by any non-identity element, such as \(kH\text{.}\) Thus any element in \(G/H\) can be written in the form \(k^iH\text{,}\) and any element in \(G\) can subsequently be written as \(k^ih\) for some \(h\in H\text{.}\) Thus \(G=KH\text{.}\)
