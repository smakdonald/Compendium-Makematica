Example 4.2.

Let \(M\) be a finitely generated \(\mathbb{Z}\)-module. Then by the FTFGAG, \(M\) is isomorphic to \(\mathbb{Z}^r \oplus \mathbb{Z}/p_1^{e_1} \oplus \cdots \oplus \mathbb{Z}/p_n^{e_n}\) for some \(r \geq 0\text{,}\) \(n \geq 0\text{,}\) primes \(p_i\) and positive integers \(e_i\text{.}\) Such a module is semisimple if and only if \(r = 0\) and \(e_i = 1\) for all \(i\text{.}\)
