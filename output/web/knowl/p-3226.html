I claim \(L = \Q(\sqrt[3]{2})\) is not a Galois extension of \(\Q\text{.}\) Let \(R\) be the set of all roots of \(x^3-2\) in \(L\text{.}\) Since \(L \subseteq \R\text{,}\) \(R\) has just one element: \(R = \{\sqrt[3]{2}\}\text{.}\) Since \(L = \Q(R)\text{,}\) the function \(\phi: \Aut(L/\Q) \to \Perm(R)\) is injective by Proposition and so since \(\# R = 1\text{,}\) we have \(\# \Aut(L/\Q) = 1\text{.}\) Thus it isn't Galois. Since it is separable, \(L\) must not be a normal extension of \(\Q\text{.}\)
