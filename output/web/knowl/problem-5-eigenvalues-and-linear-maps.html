Problem 5 â€“ Eigenvalues and Linear Maps.
Suppose that \(T : V\to V\) is a linear map, where \(V\) is a finite dimensional \(\C\)-vector space. Fix a polynomial \(p\in \C[x]\)[^1].
  1. Prove that if \(\lambda\) is an eigenvalue of \(T\) then \(p(\lambda)\) is an eigenvalue of \(p(T)\text{.}\)
  2. Prove conversely that if \(\mu\) is an eigenvalue of \(p(T)\) then there exists an eigenvalue \(\l\) of \(T\) such that \(\mu = p(\l)\text{.}\)
Proof..
Part (a).
Let \(\l\) be an eigenvalue of \(T\text{.}\) Thus there exists some \(v\) such that \(Tv=\l v\text{.}\) Notice that \(p(T)v=a_nT^nv+\dots+a_0Iv=0\) and \(p(\l)=a_n\l^n+\dots+a_0\text{.}\) Then
\begin{equation*} \begin{align}(p(T)-p(\l I))v&=p(T)v-p(\l I)v\\&=(a_nT^nv+\dots+a_0Iv)-(a_n\l^nv+\dots+a_0v)\\&=a_n(T^n-\l^n)v+\dots a_0(I-I)v\\&=0,\end{align} \end{equation*}
making \(p(\l)\) an eigenvalue of \(p(T)\text{.}\)
Part (b).
First note that if \(T\) is a scalar of the identity matrix then its only eigenvalue is \(0\text{.}\) Let \(\mu\) be an eigenvalue of \(p(T)\text{.}\) Thus \((p(T)-\mu I)v=0\) for some \(v\text{.}\) As \(\C\) is algebraically closed we can factor the polynomial \(p(x)-\mu\) into linear terms:
\begin{equation*} p(x)-\mu=c(x-r_1)\dots(x-r_n) \end{equation*}
for roots \(r_i\text{.}\) Note that \(p(r_i)-\mu=0\) for all \(r_i\text{.}\) Observe
\begin{equation*} \begin{align}(p(T)-\mu I)v&=p(T)v-(\mu I)v\\&=c(T-r_nI)\dots(T-r_1I)v\\&=0.\end{align} \end{equation*}
Thus one of these terms must be sent to zero. Note that if \(T-r_iI=0\) for any \(I\) this would make \(T\) a scaler of the identity matrix. Thus there exists some \(i\) such that \((T-r_iI)v=0\text{,}\) making \(r_i\) an eigenvalue of \(T\text{.}\) Notice that as \(p(r_i)-\mu=0\) we have \(p(r_i)=\mu\text{,}\) completing the proof.
