Proposition 3.14.

Let \(R\) be a commutative ring, and set \(I = \{r \in R | rn = 0 \;\text{for some integer}\; n\}\text{.}\) Prove that following assertions.
  1. \(I\) is an ideal in \(R\text{.}\)
  2. If \(R/I\) is a field, then each element of \(R\) is either a unit or in \(I\text{.}\)
