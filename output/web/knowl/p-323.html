To prove existence, define a function \(f: D_{2n} \to G\) by \(f(r^i) = x^i\) for \(0 \leq i \leq n-1\) and \(f(r^is) = x^is\) for \(0 \leq i \leq n-1\text{.}\) Then one may painstakenly checks that \(f(ab) = f(a)f(b)\) in each of the cases \(a = r^j, b= r^i\) or \(a = r^j, b = r^is\) or \(a = r^js, b = r^i\) or \(a = r^js, b = r^is\text{.}\) For example, using that \(sr^i = r^{-i}s\text{,}\) we have
\begin{equation*} f((r^js)(r^i)) = f(r^j(sr^i) = f(r^{j-i}s) = x^{j-i}y \end{equation*}
and
\begin{equation*} f(r^js)f(r^i)) = x^jy x^i =x^j x^{-i} y = x^{j-i} y. \end{equation*}
For the last string of equalities, we have used that, since \(yxy = x^{-1}\text{,}\) we have \(yx^i = x^{-i} y\) in \(G\) too. Note also that \(j-i\) might be negative, in which case \(r^{j-i}\) is really \(r^{n+j-i}\) and so really we would have \(f(r^{j-i}s) = x^{n+j-i}y\text{.}\) But this is OK since \(x^{j-i} = x^{n+j-i}\) too, since \(x^n = e\text{.}\)
