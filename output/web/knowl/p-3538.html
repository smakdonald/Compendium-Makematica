If f: X \to Y, C \subseteq C' \subseteq X, and C\alpha \subseteq X for all \alpha , then
  1. f(C) \subseteq f(C').
  2. f(\cup \alpha C\alpha ) = \cup \alpha f(C\alpha ).
  3. f(\cap \alpha C\alpha ) \subseteq \cap \alpha f(C\alpha ). If moreover f is injective, then f(\cap \alpha C\alpha ) = \cap \alpha f(C\alpha ).
  4. f(C' - C) \supseteq f(C') - f(C). If moreover f is injective, then f(C' - C) = f(C') - f(C).
