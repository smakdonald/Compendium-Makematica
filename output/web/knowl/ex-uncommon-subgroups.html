Example 1.62. Less Common Subgroups.

  1. \(\displaystyle R^+\leq R^\times\)
  2. The group \(L=\{1,-1,i,-i\}\) is a subgroup of \(\C^\times\text{.}\)
  3. \(\displaystyle \{1,3\}\leq \Z/8^\times\)
  4. The set of all rotations is a subgroup of \(D_{2n}\text{.}\) Similarly, the set of reflections is also a subgroup.
