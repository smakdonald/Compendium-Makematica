  1. Suppose there exist two elements \(e\) and \(e'\) such that \(ex = x = xe\) and \(e'x = x = xe'\) for all \(x\text{.}\) Then
    \begin{equation*} e = ee' = e'. \end{equation*}
    Thus only one element can satisfy this property, making the identity element unique.
  2. For a given \(x\text{,}\) if \(yx = xy = e\) and \(zx = xz = e\) fo some \(y\) and \(z\text{,}\) then \(z = ez = (yx)z = y(xz) = ye = y\text{.}\)
  3. Suppose \(xy = xz\) for some \(x,y,z\in G\text{.}\) By multiplying on both sides by \(x\inv\text{,}\) we see
    \begin{equation*} \begin{align} xy &= xz \\ x\inv(xy) &= x\inv(xz) \\ (x\inv x)y &= (x\inv x)z \\ (e)y &= (e)z \\ y &= z. \end{align} \end{equation*}
