Exercise Group 1.1.1–4. Computations and Counterexamples.

1.
Explain why each of the following is not a group.
  1. \(\displaystyle (\N,+)\)
  2. \(\displaystyle (\Z,\times)\)
  3. \((\O,+)\text{,}\) where \(\O\) is the set of odd integers.
  4. \((\Z/(n),\times)\) for all \(n<1\)
2.
Find counterexamples to each of the following (false) statements.
  1. If \(g^2=e\) then \(g=e\)
  2. If \(g^2=h^2\) then \(g=h\)
  3. For all \(g\in G\) there exists some \(h\in G\) such that \(g=h^2\text{.}\)
3.
Prove or disprove: if \(x,y\) have finite order then \(xy\) has finite order.
4.
Give an example of a pair \((G,\cdot)\) that satisfies axioms 1 and 2 of Definition 1.3 and an element of \(G\) that has a left inverse but not a right inverse.
