Definition 1.2. Monoid.

A monoid is a pair \((G, \cdot)\) where \(G\) is a set and \(\cdot\) is a Binary Operation on \(G\) satisfying the following properties:
  • Closure.
    for each \(x,y\in G\text{,}\) \(xy\in G\text{.}\)
  • Associativity.
    for all \(x,y,z\in G\) we have \((x\cdot y)\cdot z=x\cdot (y\cdot z)\)
  • Identity Element.
    there exists \(e\in G\) such that \(e·g=x·g=x\) for all \(g\in G\)
