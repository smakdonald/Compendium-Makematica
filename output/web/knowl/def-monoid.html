Definition 1.2. Monoid.

A monoid is a pair \((G, \cdot)\) where \(G\) is a set and \(\cdot\) is a Binary Operation on \(G\) satisfying the following properties:
  1. (Closure) for each \(x,y\in G\text{,}\) \(xy\in G\text{.}\)
  2. (Associativity) for all \(x,y,z\in G\) we have \((x\cdot y)\cdot z=x\cdot (y\cdot z)\)
  3. (Identity Element) there exists \(e\in G\) such that \(e·x=x·e=x\) for all \(x\in G\)
