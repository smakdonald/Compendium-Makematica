Let \(x\) be an element in \(G^{op}\text{.}\) Since \(x\) is also in \(G\text{,}\) \(G\) being a group implies that there exists an inverse element \(x\inv\) in \(G\) such that \(x 路 x\inv = x\inv 路 x = e\text{,}\) where \(e\) is the identity element of \(G\text{.}\) This implies that \(x\inv 路 x = x 路 x\inv = e\) in \(G^{op}\) as well. Therefore, every element in \(G^{op}\) has an inverse in \(G^{op}\text{,}\) satisfying the inverse element property.
