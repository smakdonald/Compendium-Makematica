Problem 2.24.

Prove the following
  1. The identity map is a group isomomorphism for any group \(G\text{.}\)
  2. The exponential map \(\exp: ({\mathbb{R}}, +) \to ({\mathbb{R}}\setminus \{0\}, \cdot)\) is a homomorphism. So is \(\ln: ({\mathbb{R}}_{>0}, \cdot) \to ({\mathbb{R}}, +)\text{.}\) In fact, these maps are inverse to each other so we obtain an isomorphism . \(({\mathbb{R}}, +)\cong ({\mathbb{R}}_{>0}, \cdot)\text{.}\)
  3. For any positive integer \(n\) and field \(F\text{,}\) the map determinant map \(\det: \operatorname{GL}_n(F) \to (F \setminus \{0\}, \cdot)\) is a group homomorphism.
