Example 11.68. Module with Infinite Basis.

Let \(R\) be any ring and \(M = R[x]\) (which, recall, is an \(R\)-module due to the evident ring map \(R \to R[x]\)). Then the countably infinite set \(\{1,x,x^2,\dots, x^n,\dots\}\) is a basis. The fact that this set is a basis is essentially part of the definition of \(R[x]\text{.}\) LemmaÂ 11.67 says that every polynomial is uniquely expressible as an \(R\)-linear combination of (a finite subset of) \(1,x,x^2,\dots\text{.}\)
