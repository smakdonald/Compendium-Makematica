Theorem 25.9. JOrdan-Holder Theorem.

Let \(M\) be a module of finite length.
  1. If \(L\subseteq M\) is a proper submodule, then \(\ell(L) < \ell(M)\text{.}\)
  2. If \(L\subseteq M\) is a nonzero submodule and \(\overline{M}=M/L\text{,}\) then \(\ell(\overline{M})< \ell(M)\text{.}\)
  3. Any filtration of \(M\) can be refined to a composition series.
  4. All strict composition series for \(M\) are equivalent, and hence have the same length.
