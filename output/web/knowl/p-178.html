Let's consider two disjoint cycles, denoted as \((a1 a2 ... am)\) and \((b1 b2 ... bn)\text{,}\) where \(ai\)'s and \(bi\)'s are distinct elements. Let's analyze the composition \((a1 a2 ... am) ∘ (b1 b2 ... bn)\text{:}\) When we apply the composition to an element \(ai\text{,}\) we have:
\begin{equation*} (a1 a2 ... am) ∘ (b1 b2 ... bn)(ai) = (a1 a2 ... am)(b1 b2 ... bn(ai)) = (a1 a2 ... am)(ai) = ai+1 (\text{ if } ai ≠ am) \text{ or }a1 (\text{ if }ai = am) \end{equation*}
Similarly, when we apply the composition to an element \(bj\text{,}\) we have:
\begin{equation*} (b1 b2 ... bn) ∘ (a1 a2 ... am)(bj) = (b1 b2 ... bn)(a1 a2 ... am(bj)) = (b1 b2 ... bn)(bj) = bj+1 (\text{ if } bj ≠ bn) \text{ or }b1 (\text{ if } bj = bn) \end{equation*}
From these calculations, we can observe that the composition \((a1 a2 ... am) ∘ (b1 b2 ... bn)\) maps each element \(ai\) to \(ai+1\) (if \(ai ≠ am\)), and it maps \(am\) to \(a1\text{.}\) Similarly, the composition \((b1 b2 ... bn) ∘ (a1 a2 ... am)\) maps each element \(bj\) to \(bj+1\) (if \(bj ≠ bn\)), and it maps \(bn\) to \(b1\text{.}\) Now, let's consider the composition \((b1 b2 ... bn) ∘ (a1 a2 ... am)\) and evaluate its effect on the elements ai and am:
\begin{equation*} (b1 b2 ... bn) ∘ (a1 a2 ... am)(ai) = (b1 b2 ... bn)(a1 a2 ... am(ai)) = (b1 b2 ... bn)(ai) = ai+1 (\text{ if } ai ≠ am) \text{ or } a1 (\text{ if } ai = am) \end{equation*}
\begin{equation*} (b1 b2 ... bn) ∘ (a1 a2 ... am)(am) = (b1 b2 ... bn)(a1 a2 ... am(am)) = (b1 b2 ... bn)(a1 a2 ... am) = am+1 (\text{ if } am ≠ a1) \text{ or } a2 (\text{ if } am = a1) \end{equation*}
Comparing these results with the previous composition, we see that the effects on \(ai\) and \(am\) are the same in both compositions. This implies that the compositions \((a1 a2 ... am) ∘ (b1 b2 ... bn)\) and \((b1 b2 ... bn) ∘ (a1 a2 ... am)\) are identical. Therefore, we have shown that disjoint cycles commute, meaning that the order in which they are composed does not affect the final result.
