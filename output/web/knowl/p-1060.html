The following are ring isomorphism invariants:
  1. all group isomorphism invariants of the additive group, including the isomorphism class (i.e., if \(R\cong S\) then \((R,+)\cong(S,+)\)).
  2. being unital, being commutative, division ring, field, integral domain
  3. the number of zerodivisors.
  4. if \(R\) is unital, all group isomorphism invariants of the group of units, including the isomorphism class (i.e., if \(R\cong S\) then \((R^\times,\cdot)\cong(S^\times,\cdot)\)).
  5. the isomorphism type of the center (i.e., if \(R\cong S\) then \(Z(R)\cong Z(S)\)).
