Item 3.

Prove the converse to the previous part is false, as follows: Let \(F\) be a field and \(R = F[x_1, x_2, \dots]\text{,}\) the ring of polynomials in the infinite list of variables \(x_1, x_2, \dots\text{.}\) (So, an element of \(R\) is a {} \(F\)-linear combination of monomials of the form \(x_1^{e_1} \cdots x_n^{e_n}\) for \(n \geq 0\) and \(e_i \geq 1\text{.}\) Note that each element of \(R\) involves only a finite number of variables, but there is no uniform bound on how many such variables can be involved in the elements of \(R\text{.}\)) You may assume without proof that \(R\) is a ring with the usual rules for adding and multiplying polynomials, which make sense since each element of \(R\) involves only a finite number of variables. Finally let \(I\) be the ideal of \(R\) generated by the variables \(x_1, x_2, \dots\text{.}\) Prove \(R\) is finitely generated as an \(R\)-module but the submodule \(I\) is not.
