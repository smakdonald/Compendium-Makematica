Proof

We start by recalling that a function \(\alpha: X \to Y\) between two sets is bijective if and only if there is a function \(\beta: Y \to X\) such that \(\alpha\circ \beta= \operatorname{id}_Y\) and \(\beta\circ \alpha= \operatorname{id}_X\text{.}\)
The \(\Leftarrow\) is thus immediate. For the \(\Rightarrow\) direction, if \(f\) is bijective homomorphism, then it certainly has a set-theoretic two-sided inverse \(g\text{.}\) But we need to show \(g\) is actually a homomorphism: for \(x,y \in H\) we have \(f(g(xy)) = xy = f(g(x))f(g(y)) = f(g(x)g(y))\text{.}\) Since \(f\) is one-to-one, \(g(xy) = g(x)g(y)\text{.}\)
