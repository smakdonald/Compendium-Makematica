Prove that if \(R\) is a commutative ring with \(1\neq 0\) then \(R^m \cong R^n\) as \(R\)-modules if and only if \(m = n\text{.}\) In order to do that, you will complete the following steps:
  1. Show that if \(I\) is any ideal of \(R\) and \(M\) is any \(R\)-module, then \(M/IM\) is an \(R/I\)-module via
    \begin{equation*} (r + I) \cdot (m + IM) = rm + IM. \end{equation*}
  2. Show that if \(I\) is any ideal of \(R\text{,}\) then \(R^n/IR^n \cong (R/I)^n\) as \(R/I\)-modules.
  3. Apply the previous part when \(I = \fm\) is a maximal ideal of \(R\text{.}\) You will need to use the following fact, which we shall prove in class very soon: if \(F\) is a field, then \(F^n\cong F^m\) as \(F\)-vector spaces if and only if \(m=n\text{.}\)
