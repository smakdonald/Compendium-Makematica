Theorem 2.20. Isomorphism Preservations.

Let \(G\) and \(H\) be groups. If \(\varphi:G\to H\) is an isomorphism, then the following hold:
  1. \(\displaystyle |G|=|H|\)
  2. \(|g|=|\varphi(g)|\) for all \(g\in G\) and \(|h|=|\varphi\inv(h)|\) for all \(h\in H\)
  3. \(G\) is abelian if and only if \(H\) is abelian
  4. \(\displaystyle |Z(G)|=|Z(H)|\)
  5. \(G\) is finitely generated if and only if \(H\) is finitely generated.
