Summary 3.3.

  • Given \(N\nsg G\text{,}\) the Quotient Group \(G/N\) is the set of left cosets of \(N\) in \(G\) with multiplication given by \(xN\cdot yN=(xy)N\text{.}\) This is a well defined group if and only if \(N\) is normal.  2 
  • A subgroup is normal if and only if it is the kernel of a homomorphism.  3 
  • If \(G\) is finite we have \(\left | G/N\right |=\frac{|G|}{|N|}\text{,}\) and the Quotient Map is a surjective homomorphism.  4 
  • If \(G/Z(G)\) is cyclic, then \(G\) is abelian.  5
