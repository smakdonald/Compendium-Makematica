Note that \(|xHx\inv| = |H|\) for all \(x \in G\text{.}\) Also, by the previous problem, the number of distinct subgroups \(xHx\inv\) is \([G : NG(H)]\text{.}\) Let \(n = [G : N_G(H)]\) and \(x_iHx\inv_i\) for \(i = 1,\dots,n\) the distinct conjugates of \(H\text{.}\) Note first that if \(n = 1\) then \(H \nsg G\) and result holds since \(H \neq G\text{.}\) So assume \(n\geq2\text{.}\) Since each conjugate contains \(|H|-1\) non-identity elements, we have
\begin{equation*} \bigcup_{x\in G} xHx\inv|= | \bigcup^n_{i=1} x_iHx\inv_i |\geq1 + \sum^n_{i=1} (|H|\inv) = 1 + n|H|-n. \end{equation*}
As \(n = [G : N_G(H)] \leq[G : H]\) (as \(H \sse N_G(H)\)), we have \(n|H|≤|[G : H]|H| = |G|\text{.}\) Thus, as \(n ≥2, 1 + n|H|-n ≤1 + |G|-n < |G|.\)
