  1. \(\Z\) is a UFD by the Fundamental Theorem of Arithmetic.
  2. \(F[x]\) where \(F\) is a field is a UFD. This is the case because \(F[x]\) is a Euclidean domain and Euclidean domains are UFD’s (we’ll prove this shortly).
  3. We will eventually prove that if \(R\) is a UFD then so is \(R[x]\text{.}\) It follows that \(F[x_1, \dots, x_n]\) is a UFD for all \(n\text{.}\) Note that if \(n > 1\text{,}\) this ring is not a PID and hence not a Euclidean domain.
