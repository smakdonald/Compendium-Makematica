Let \(-*-\) be a binary operation on a set \(S\text{.}\)
  1. An operation satisfies the associative property if for all \(x,y,z\in S\) we have \((x*y)*z=x*(y*z)\text{.}\)
  2. An operation satisfies the identity property if there exists \(e\in G\) such that \(e*g=g*e=g\) for all \(g\in G\text{.}\) Such an element \(e\) is called an identity element.
  3. An operation satisfies the inverse property if for each \(x\in S\text{,}\) there is an element \(y\in S\) such that \(x * y=e=y * x\text{,}\) where \(e\) is an identity element of \(S\text{.}\) Such an element \(y\) is called an inverse of the element \(x\text{.}\)
