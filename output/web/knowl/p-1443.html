Suppose that \(\alpha:M_1 \to M_2\) is an isomorphism of \(K[G]\)-modules. Identifying \(M_1=M_2=K\) as vector spaces, we have \(\alpha(k) = ck\) for some \(c\neq 0\text{.}\) Then,
\begin{equation*} c \rho_1(g) (k) = \alpha \rho_1(g) (k) = \rho_2(g) \alpha(k) = \rho_2(g) (ck) = c \rho_2(g) (k) \end{equation*}
for all \(k\in K\text{,}\) so \(\rho_1(g)(k)=\rho_2(g)(k)\) for all \(k\in K\text{.}\)
