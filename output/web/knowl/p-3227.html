By Proposition , for each \(g \in G\) we have \(g(\a) \in R\text{.}\) Moreover, by definition of \(H\text{,}\) for all \(h \in H\text{,}\) we have \(h(\a) = \a\) and hence \(gh(\a)=g(h(\a))=g(\a)\text{.}\) This proves \(\theta\) is a well-defined function (i.e., it is independent of left coset representative). For \(g_1, g_2 \in G\text{,}\) if \(g_1(\a) = g_2(\a)\) then \(g_2^{-1}g_1(\a) = \a\) which implies \(g_2^{-1}g_1 \in H\) (since if an automorphism of \(L\) fixes \(F\) and \(\a\) then it fixes \(F(\a)\)). Thus \(g_1(\a) = g_2(\a)\) implies \(g_1H = g_2H\) and hence \(\theta\) is one-to-one.
