Given an \(R[G]\)-module structure on \(V\text{,}\) for every \(g\in G\text{,}\) there is a map \(m_g: V\to V\) given by \(v\mapsto g\cdot v\text{.}\) We have \(m_g (rv) = g (rv) = rg (v) = r m_g(v)\text{,}\) so \(m_g\) is \(R\)-linear. Moreover, the map \(\rho:G\to \operatorname{End}_R(V)\) that sends \(g\mapsto m_g\) preserves multiplication and identity: \(\rho(gh)(v) = gh v = g(hv) = \rho(g) \rho(h) (v)\) and \(\rho(e)(v) = v\text{.}\) Thus, we obtain an \(R\)-linear representation \(\rho:G\to \operatorname{Aut}_R(V)\text{.}\)
