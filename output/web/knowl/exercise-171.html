Exploration 17.34.

Let \(A\) and \(B\) be \(n \times n\) matrices with entries in \(\Q\text{.}\) Prove \(A\) and \(B\) are similar in \(\Mat_{n \times n}(\Q)\) if and only if \(A\) and \(B\) are similar in \(\Mat_{n \times n}(\R)\text{.}\) (That is, show there is a \(P \in GL_n(\Q)\) such that \(A = PBP^{-1}\) if and only if there is a \(R \in GL_n(\R)\) such that \(A = RBR^{-1}\text{.}\)) {}: Use the Theorem on Rational Canonical Forms.
