  • Magma.
    A pair \((G, \cdot)\) is called a magma.
  • Semigroup.
    A semigroup is a pair \((G, \cdot)\) where \(G\) is a set and \(\cdot\) is a Binary Operation on \(G\) satisfying the associative property.
  • Monoid.
    A monoid is a pair \((G, \cdot)\) where \(G\) is a set and \(\cdot\) is a binary operation on \(G\) satisfying the associative and identity properties.
  • Group.
    A group is a pair \((G, \cdot)\) where \(G\) is a set and \(\cdot\) is a binary operation on \(G\) satisfying the associative, identity, and inverse properties.
  • Abelian Group.
    An abelian group is a pair \((G, \cdot)\) where \(G\) is a set and \(\cdot\) is a binary operation on \(G\) satisfying the associative, identity, inverse, and commutative properties.
