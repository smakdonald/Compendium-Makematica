Solution 5.2.7.1.
Let \(G\) be a finite group, \(p\) a prime dividing the order of \(G\text{,}\) and \(H\) a normal subgroup of order \(pm\) for some \(m > 0\text{.}\)
As \(H\) is normal we have \(gHg\inv=H\) for all \(g\in G\text{.}\) As it is a \(p\) group it is contained in some Sylow \(p\)-subgroup. All Sylow \(p\)-subgroups are conjugate to each other, but \(H\) is fixed by conjugation so its actually in all of them.
Consider \(Z(H)\text{,}\) which is abelian in \(H\text{.}\) The center of \(p\)-groups are nontrivial so that's good. Let \(g\in G\text{,}\) \(h\in Z(H)\text{,}\) and consider \(ghg\inv\text{.}\) As \(H\nsg G\) we know \(ghg\inv\in H\text{.}\) Let \(h'\in H\text{.}\) We see \(ghg\inv h'=ghh''g\inv\text{,}\) since \(H\) is still normal. \(h\in Z(G)\) so we have \(gh''hg\inv\text{.}\)
Notice that \(h''g\inv=g\inv h'\text{,}\) and thus \(h'=gh''g\inv\text{.}\) Thus \(gh''=h'g\text{,}\) and so we can substitute the last line in the previous paragraph to see \(h'ghg'\inv\text{.}\) Thus we have normality!
