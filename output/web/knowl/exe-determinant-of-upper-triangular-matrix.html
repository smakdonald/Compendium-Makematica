Example B.42. Determinant of Upper Triangular Matrix.

If \(A\) is upper triangular, then \(\det(A) = \prod_i a_{i,i}\text{.}\) For note that for such a matrix the only non-zero terms in the formula for \(\det(A)\) occur when \(\s(i) \leq i\) for all \(i\text{,}\) and this can only happen when \(\s = \id\text{.}\) Similarly, if \(A\) is lower triangular, then \(\det(A) = \prod_i a_{i,i}\text{.}\)
In particular, if \(E\) is a Type I elementary matrix, then \(\det(E) = 1\) and if \(E\) is a type II elementary matrix with scalar \(u \ne 0\text{,}\) then \(\det(E) = u\text{.}\) We'll look at the determinants of Type III elementary matrices later.
An even more special case is \(\det(I_n) = 1\text{.}\)
