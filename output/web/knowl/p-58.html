Let \(G\) be a group.
  1. Unique Identity.
    The element \(e\) satisfying the Identity Axiom of Definition 1.2 is unique, and we thus refer to it as the identity element of \(G\text{.}\)
  2. Unique Inverses.
    For each \(x\in G\text{,}\) the element satisfying the Inverse Axiom of Definition 1.9 is unique, and we thus refer to it as the the inverse of \(x\text{.}\)
  3. Cancellation.
    Suppose \(x,y,z\in G\text{.}\) If \(xy = xz\) then \(y=z\text{.}\) Similarly, if \(yx = zx\) we have \(y = z\text{.}\)
