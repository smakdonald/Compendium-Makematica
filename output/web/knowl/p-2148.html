If \(f: X \to Y, E \subseteq E' \subseteq Y\text{,}\) and \(E_\alpha \subseteq Y\) for all \(\alpha\text{,}\) then
  1. \(f\inv(E) \subseteq f\inv(E')\text{.}\)
  2. \(f\inv(\bigcup_\alpha E_\alpha ) = \bigcup_\alpha f\inv(E_\alpha )\text{.}\)
  3. \(f\inv(\bigcap_\alpha E_\alpha ) = \bigcap_\alpha f\inv(E_\alpha )\text{.}\)
  4. \(f\inv(E' \sm E) = f\inv(E') \sm f\inv(E)\text{.}\)
