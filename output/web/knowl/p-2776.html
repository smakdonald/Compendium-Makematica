If \(A\) is upper triangular, then \(\det(A) = \prod_i a_{i,i}\text{.}\) For note that for such a matrix the only non-zero terms in the formula for \(\det(A)\) occur when \(\s(i) \leq i\) for all \(i\text{,}\) and this can only happen when \(\s = \id\text{.}\) Similarly, if \(A\) is lower triangular, then \(\det(A) = \prod_i a_{i,i}\text{.}\)
