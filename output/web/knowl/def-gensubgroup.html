Definition 3.18. Generated Subgroup.

Given a group \(G\) and a subset \(S\) of \(G\text{,}\) the subgroup of \(G\) generated by \(S\) is
\begin{equation*} \langle S \rangle := \bigcap_{H \leq G, H \supseteq X} H. \end{equation*}
If \(S=\{s\}\) is a set with one element then we write \(\langle S \rangle=\langle s \rangle\) and we refer to this as the cyclic subgroup generated by \(s\text{.}\)
