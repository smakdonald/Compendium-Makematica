Anequivalence relation\sim on a set \(X\) is a subset \(S of X \times X\) (where \((a,b) \in S\) is written \(a \sim b\)) that satisfies the following for all \(a,b,c\) in \(X\text{:}\) (1) a \sim a (reflexive), (2) a \sim b implies b \sim a (symmetric), and (3) a \sim b and b \sim c implies a \sim c (transitive).
