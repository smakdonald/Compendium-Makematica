This mostly follows from Artin-Wedderburn and Maschke's Theorem. What needs to be noted is that each division ring here contains a copy of \(K\) in its center. Indeed, we recall that each \(D_i\) is given as the opposite ring of \(\operatorname{End}_{K[G]}(M_i)\) for some simple module \(M_i\text{.}\) For \(\lambda\in K\text{,}\) we have the map \(M_i \xrightarrow{\lambda \cdot} M_i\) which commutes with any \(K[G]\)-linear map from \(M_i\) to itself.
