Let \(f \in I\text{.}\) The leading coefficient of \(f\) is an \(R\)-linear combination of \(a_1, \ldots, a_t\text{.}\) If \(f\) has degree greater than \(N\text{,}\) then we can cancel off the leading term of \(f\) by subtracting a suitable combination of the \(f_i\text{.}\) Therefore, any \(f \in I\) can be written as \(f = g+ h\) for some \(h \in (f_1, \ldots, f_t)\) and \(g \in I\) with degree at most \(N\text{.}\) In particular, note that
\begin{equation*} g \in I \cap \frac{R + Rx + \cdots + R x^N}. \end{equation*}
Since \(I \cap (R + Rx + \cdots + R x^N)\) is a submodule of the finitely generated free \(R\)-module \(R + Rx + \cdots + R x^N\text{,}\) it must also be finitely generated as an \(R\)-module[^3]. Given such a generating set, say \(I \cap (R + Rx + \cdots + Rx^N) = (f_{t+1}, \ldots, f_s)\text{,}\) we can write any element \(f \in I\) as an \(R[x]\)-linear combination of these generators \(f_{t+1}, \ldots, f_s\) and the original \(f_{1}, \ldots, f_t\text{.}\) Therefore, \(I = (f_1, \ldots, f_t, f_{t+1}, \ldots, f_s)\) is finitely generated as an ideal in \(R[x]\text{,}\) and \(R[x]\) is a noetherian ring.
