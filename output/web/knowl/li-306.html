Item 4.b.
Suppose there exists some \(k\in K\) such that \(k\not\in H\text{.}\) We examine \(kH\in G/H\text{.}\) Notice that since \(G/H\) has prime order it is cyclic, and thus generated by any non-identity element, such as \(kH\text{.}\) Thus any element in \(G/H\) can be written in the form \(k^iH\text{,}\) and any element in \(G\) can subsequently be written as \(k^ih\) for some \(h\in H\text{.}\) Thus \(G=KH\text{.}\)
