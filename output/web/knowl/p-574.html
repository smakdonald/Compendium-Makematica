If such a \(\overline{f}\) exists, it is necessarily unique since \(G \twoheadrightarrow G/N\) is onto. In fact, if \(f=\pi\circ \overline{f}\) then \(\overline{f}\) has to be given by the formula
\begin{equation*} \overline{f}(gN) = f(g). \end{equation*}
We now need to show that this formula determines a well-defined homomorphism: If \(xN = yN\text{,}\) then \(y^{-1}x \in N \subseteq \operatorname{Ker}(f)\) and so \(f(y)^{-1} f(x) = e\text{,}\) whence \(f(y) = f(x)\text{.}\) For any \(x,y\) we have
\begin{equation*} \overline{f}(xNyN) = \overline{f}(xyN) = f(xy) = f(x)f(y) =\overline{f}(xN) \overline{f}(yN). \end{equation*}
