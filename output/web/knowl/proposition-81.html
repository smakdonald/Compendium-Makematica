Proposition 3.24.

Suppose \(A \in \Mat_{n \times n}(F)\) where \(F\) is a field and \(A^k = 0\) for some integer \(k \geq 1\text{.}\) Prove \(A^n = 0\text{.}\) {}: Note that for any \(m \geq 1\text{,}\) \(\im(T_A^{m+1}) \subseteq \im(T_A^{m})\text{.}\) Show that if equality holds then \(A^m = 0\text{.}\)
