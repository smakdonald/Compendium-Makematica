To show the uniqueness, say \(C = \{c_1, \dots, c_n\}\text{,}\) say \(C' = \{c'_1, \dots, c'_n\}\) are ordered bases such that \([\id_V]_B^{C'} = P = [\id_V]_B^C\text{.}\) Then
\begin{equation*} [\id_V]_C^{C'} = [\id_V]_B^{C'} [\id_V]_C^B = [\id_V]_B^{C'} \left( [\id_V]_B^{C} \right)^{-1} = PP^{-1} = I_n. \end{equation*}
For each \(i\) we have
\begin{equation*} [c_i]^C = (0, \dots, 0, 1, 0, \dots, 0)^\tau \end{equation*}
and
\begin{equation*} [c_i]^C = I_n [c_i]^C = [\id_V]_C^{C'} [c_i]^C = [c_i]^{C'} \end{equation*}
so that
\begin{equation*} [c_i]^{C'} = (0, \dots, 0, 1, 0, \dots, 0)^\tau \end{equation*}
which gives \(c_i = c_i'\text{.}\)
