For a left ideal \(I\text{,}\) the submodules of \(R/I\) are in bijective correspondence with the left \(R\)-submodules of \(R\) that contain \(I\text{,}\) i.e., the left ideals that contain \(I\text{.}\) It is then clear that if \(I\) is a maximal left ideal, then \(R/I\) is simple, so (3) implies (1). On the other hand, if \(M\) is simple then it is cyclic (since (1) implies (2)), so \(M\cong R/I\) for some left ideal \(I\text{,}\) and if \(I \subsetneqq J\) for some proper left ideal \(J\text{,}\) then \(0 \neq J/I \subsetneqq R/I\text{;}\) thus (1) implies (3).
