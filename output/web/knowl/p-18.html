  1. (Unique Identity) In a semigroup \(G\text{,}\) the element \(e\) satisfying the second axiom of Definition 1.2 is unique, and we thus refer to it as the identity element of \(G\text{.}\)
  2. (Unique Inverses) In a group \(G\text{,}\) for each \(x\text{,}\) the element \(y\) satisfying the additional axiom of Definition 1.5 is unique, and we thus refer to it as the the inverse of \(x\) and we will henceforth denote it by \(x^{-1}\text{.}\)
  3. (Cancellation) In a group \(G\text{,}\) if \(xy = xz\) then \(y=z\text{.}\) Similarly, if \(yx = zx\) we have \(y = z\text{.}\)
