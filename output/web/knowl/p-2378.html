First, we show that the center of \(G\) is nontrivial. Suppose by way of contradiction that \(Z(G)=\{e\}\text{.}\) We examine the conjugacy classes of \(G\text{.}\) From the Class Equation, we know
\begin{equation*} |G| = |Z(G)| + \sum_i^r [G : C_G(g_i)]. \end{equation*}
Note that \(|G|=p^n\) for some \(n\in\N\text{,}\) meaning that the only divisors of \(|G|\) are powers of \(p\text{.}\) In finite groups, each conjugacy class must divide the order of the group by TheoremÂ 5.42. By Lagrange's Theorem, \([G : C_G(g_i)]=\frac{|G|}{|C_G(g_i)|}=\frac{p^n}{p^m}=p^k\text{,}\) where \(m,k\in\N\text{.}\) Since \(|Z(G)|=1\text{,}\) we see that \(\sum_i^r |G : C_G(g_i)|\bigg|p^n-1\text{,}\) which is impossible given that \(p^{n-1} \not\mid p^n\text{.}\) Thus \(Z(G)\neq\{e\}\text{.}\)
