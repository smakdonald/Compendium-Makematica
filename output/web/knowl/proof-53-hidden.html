If \(m := \ell(M)\text{,}\) consider a strict composition series of \(M\) of length \(m\text{,}\) say
\begin{equation*} M_\bullet: \qquad 0 = M_0 \subsetneqq M_1 \subsetneqq M_2 \subsetneqq \cdots \subsetneqq M_m = M. \end{equation*}
  1. Consider the filtration
    \begin{equation*} L_\bullet: \qquad 0 = M_0 \cap L \subseteq M_1 \cap L \subseteq M_2 \cap L \subseteq \cdots \subseteq M_m \cap L = L. \end{equation*}
    By the Second Isomorphism Theorem, its composition factors satisfy
    \begin{equation*} \frac{M_{i+1} \cap L}{M_i \cap L} = \frac{M_{i+1} \cap L}{(M_{i+1} \cap L) \cap M_i} \cong \frac{M_{i+1} \cap L + M_i} {M_i}. \end{equation*}
    The right hand side is a submodule of \(M_{i+1}/M_i\text{,}\) which by assumption is simple, so our filtration is in fact a composition series of length \(n\text{.}\) Then for any \(i\) either
    \begin{equation*} \frac{M_{i+1} \cap L}{M_i \cap L} = 0 \quad \textrm{ or } \quad \frac{M_{i+1} \cap L}{M_i \cap L}\cong \frac{M_{i+1}}{M_i}. \end{equation*}
    We claim that the latter case does not hold for all \(i\text{:}\) if it did, we would have \(0=M_0 = M_0 \cap L\text{,}\) and inductively \(M_{i+1} \cap L = M_{i+1}\) for all \(i\) and in particular for \(i=m-1\text{,}\) we have \(M = M\cap L\text{,}\) contradicting that \(L\) is proper. Thus, for some \(i\text{,}\) the first case holds. We can then skip that \(i\) and obtain a composition series of length less than \(n\text{,}\) so \(\ell(L)<m\text{.}\)
  2. Consider the filtration
    \begin{equation*} \overline{M}_\bullet: \qquad 0 = \frac{M_0 + L }{L} \subseteq \frac{M_1 + L }{L} \subseteq \cdots \subseteq \frac{M_n + L }{L} = \overline{M}. \end{equation*}
    The factors satisfy
    \begin{equation*} \frac{(M_{i+1} + L )/ L }{(M_i + L) / L} \cong \frac{M_{i+1} + L}{M_{i} + L} \cong \frac{M_{i+1} + (M_i+L)}{M_i+L}\cong \frac{M_{i+1} }{M_{i+1} \cap( M_i + L)}, \end{equation*}
    and since \(M_i \subseteq M_{i+1} \cap (M_i+L)\text{,}\) these are quotient modules of the simple module \(M_{i+1}/M_i\text{,}\) so this is a composition series. Then for any \(i\) either
    \begin{equation*} \frac{(M_{i+1} + L )/ L }{(M_i + L) / L} = 0 \quad \textrm{ or } \quad \frac{M_{i+1} }{M_{i+1} \cap( M_i + L)} \cong \frac{M_{i+1}}{M_i}. \end{equation*}
    We claim that the latter case does not hold for all \(i\text{:}\) if it did, we would have then \(M_{i+1} \cap (M_i+L) = M_{i}\) for all \(i\text{,}\) so
    \begin{equation*} \frac{M_{i+1} + L}{M_{i+1}} \cong \frac{L+M_i}{(L+M_i) \cap M_{i+1}} =\frac{L+M_i}{M_i} \end{equation*}
    for all \(i\text{,}\) and hence \(L \cong (L+M_0)/M_0 \cong (L+M_n)/M_n \cong 0\text{,}\) contradicting that \(L\neq 0\text{.}\) Thus, for some \(i\text{,}\) the first case holds, and we can skip that \(i\) to obtain a composition series of length less than \(n\text{,}\) so \(\ell(\overline{M})<m\text{.}\)
  3. We proceed by induction on length again. Given a filtration of \(M\text{,}\) we can suppose that there is some nonzero proper submodule \(L\) in the filtration, since otherwise we could just take any composition series. Then \(L\) and \(\overline{M}\) has length less than \(M\text{.}\) The filtration up to \(L\) can be refined to a strict composition series by the induction hypothesis, and the filtration from \(L\) to \(M\) taken mod \(L\) can be refined to a strict composition series for \(\overline{M}\text{;}\) pulling back as in the remark above, we get the strict composition series we want.
  4. We show by induction on \(m\) that for any module of length \(m\text{,}\) all of its strict composition series are equivalent. Assume that \(\ell(M)=m\text{.}\) If \(m=1\text{,}\) the claim is clear since we are dealing with a simple module. Suppose that
    \begin{equation*} N_\bullet: \qquad 0 = N_0 \subsetneqq N_1 \subsetneqq \cdots \subsetneqq N_n = M \end{equation*}
    is another strict composition series for \(M\text{,}\) so \(n\geq m\text{.}\) If \(N_{n-1}=M_{m-1}\text{,}\) then since \(\ell(M_{m-1})\leq m-1\) the two composition series we have for \(M_{m-1}\) are equivalent by induction, so the two given series are equivalent.
    If \(N_{n-1} \neq M_{m-1}\text{,}\) since \(M/M_{m-1}\) is simple, \(M_{m-1}\) is not properly contained in \(N_{n-1}\text{,}\) so the image of \(M_{m-1}\) in \(M/N_{n-1}\) is nonzero, so equals all of \(M\text{,}\) which means that \(N_{n-1}+M_{m-1} = M\text{.}\) Set \(K=N_{n-1}\cap M_{m-1}\text{.}\) By the second isomorphism theorem, we then have
    \begin{equation*} \frac{M}{M_{m-1}} = \frac{M_{m-1}+N_{n-1}}{M_{m-1}} \cong \frac{N_{n-1}}{K} \end{equation*}
    and similarly \(M/N_{n-1} \cong M_{m-1} / K\text{,}\) and both of these modules are simple.
    Fix a strict composition series for \(K\text{:}\)
    \begin{equation*} K_{\bullet}: \qquad 0 = K_0 \subsetneqq K_1 \subsetneqq \cdots \subsetneqq K_k = K \end{equation*}
    and extend to a strict composition series for \(M_{m-1}\text{:}\)
    \begin{equation*} K'_{\bullet}: \qquad 0 = K_0 \subsetneqq K_1 \subsetneqq \cdots \subsetneqq K_k = K \subsetneqq M_{m-1}. \end{equation*}
    Since we also have the strict composition series
    \begin{equation*} M_{\bullet \leq m-1} : \qquad 0 = M_0 \subsetneqq M_1 \subsetneqq M_2 \subsetneqq \cdots \subsetneqq M_{m-1} \end{equation*}
    of length \(m-1\text{,}\) we must have that \(k=m-2\) and \(K'_{\bullet}\) is equivalent to \(M_{\bullet \leq n-1}\text{.}\) Thus, the composition factors of \(M_{\bullet \leq m-1}\) are those of \(K\) plus one copy of \(M_{m-1} / K \cong M/N_{n-1}\text{.}\)
    Now,
    \begin{equation*} K''_{\bullet}: \qquad 0 = K_0 \subsetneqq K_1 \subsetneqq \cdots \subsetneqq K_{m-2} = K \subsetneqq N_{n-1} \end{equation*}
    is a strict composition series for \(N_{n-1}\text{,}\) so \(n=m\text{.}\) Then, \(K''_{\bullet}\) is equivalent to the strict composition series
    \begin{equation*} N_{\bullet \leq n-1}: \qquad 0 = N_0 \subsetneqq N_1 \subsetneqq \cdots \subsetneqq N_{n-1}. \end{equation*}
    Thus, the composition factors of \(N_{\bullet \leq n-1}\) are those of \(K\) plus one copy of \(N_{n-1} / K \cong M/M_{n-1}\text{.}\)
    It follows that the composition series \(M_\bullet\) and \(N_\bullet\) are equivalent.
