For a two-sided ideal \(I\) of \(R\text{,}\) the set of additive cosets modulo \(I\) is \(R/I=\{r+I : r\in R\}.\) This is an abelian group with respect to addition given by \((r + I) + (s + I) = (r +s ) + I\text{.}\) The quotient ring of \(R\) modulo \(I\) is the set \(R/I\) with addition defined as above and multiplication given by \((r + I) \cdot (s + I) = (rs) + I.\)
