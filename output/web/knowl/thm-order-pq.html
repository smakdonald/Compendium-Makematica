Theorem 8.10. Classification for Groups of Order \(pq\).

Let \(p<q\) be primes.
  1. If \(p\nmid q-1\) there is a unique group of order \(pq\) up to isomorphism, namely \(C_{pq}\text{.}\)
  2. If \(p\mid q-1\) there are exactly two groups of order \(pq\) up to isomorphism, namely \(C_{pq}\) and a non-abelian group.
