Problem 2.16.

  • In the commutative ring \(\Z\text{,}\) we have \((2,3)=(1)=\Z\text{.}\) Indeed any element \(n\in \Z\) can be written as \(n=(-n)\cdot 2+n\cdot 3=n\cdot 1\text{.}\) Note that \(1=\gcd(2,3)\text{.}\)
  • In the commutative ring \(\Z\text{,}\) we have \((2,4)=(2)=2\Z\text{,}\) the set of all even integers. Notice this shows that different sets can generate the same ideal. Also note that \(2=\gcd(2,4)\text{.}\)
  • In \(\Z[x]\text{,}\) we have \((2,x)=\{a+xp(x) \mid a \;\text{is even}, p(x)\in\Z[x]\}\) and this ideal cannot be generated by a single element.
