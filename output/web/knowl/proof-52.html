Proof

For the first assertion, let \(f:M\to N\) be \(R\)-linear and nonzero. Then \(\operatorname{ker}(f)\neq M\text{,}\) so \(\operatorname{ker}(f)=0\) by simplicity, and \(\operatorname{im}(f) \neq 0\text{,}\) so \(\operatorname{im}(f) = N\text{.}\)
For the second, recall that \(\operatorname{End}_R(M)\) is a ring. If \(f\in \operatorname{End}_R(M)\) is nonzero, then by the first part, it is an isomorphism, so it has a two-sided inverse in \(\operatorname{End}_R(M)\text{.}\)
