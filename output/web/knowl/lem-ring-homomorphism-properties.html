Lemma 7.56. Properties of Ring Maps.

If \(f: R \to S\) is a ring homomorphism, then
  1. \(f(0_R) = 0_S\) and \(f(-x)=-f(x)\text{.}\)
  2. if \(R\text{,}\) \(S\) are unital then \(f(1_R)\) can be either \(0_S, 1_S\) or a zerodivisor.
  3. If \(f(1_R)=1_S\) and \(u\in R^\times\) then \(f(u^{-1})=f(u)^{-1}\text{.}\)
  4. If \(f: R \to S\) and \(g:S\to T\) are ring homomorphisms (or isomorphisms, respectively), then \(g\circ f:R\to T\) is a ring homomorphism (or isomorphism).
