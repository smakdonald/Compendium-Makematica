The \(\Leftarrow\) is thus immediate. For the \(\Rightarrow\) direction, if \(\varphi\) is bijective homomorphism, then it certainly has a set-theoretic two-sided inverse \(\phi\text{.}\) But we need to show \(\phi\) is actually a homomorphism: for \(x,y \in H\) we have \(\varphi(\phi(xy)) = xy = (\phi(x))\varphi(g(y)) = \varphi(\phi(x)\phi(y))\text{.}\) Since \(\varphi\) is one-to-one, \(\phi(xy) = \phi(x)\phi(y)\text{.}\)
