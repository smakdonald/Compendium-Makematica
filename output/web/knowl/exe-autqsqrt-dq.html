\(\exe\) - \(\Aut(\Q(\sqrt d)/\Q)\).

For any square-free integer \(d\text{,}\) \(\Aut(\Q(\sqrt{d})/\Q)\) has order \(2\text{,}\) and its two elements are the identity and the map sending \(a + b \sqrt{d}\) to \(a - b \sqrt{d}\text{.}\) Checking that each really is an element of this group and that there are the only two elements in this group is done similarly to the previous example.
