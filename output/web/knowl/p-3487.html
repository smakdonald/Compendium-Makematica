Let \(f: X \to Y\text{.}\)
  1. If \(A \subseteq X\text{,}\) then \(A \subseteq f\inv(f(A))\text{;}\) if moreover \(f\) is injective then \(A = f\inv(f(A))\text{.}\)
  2. If \(B \subseteq Y\) then \(f(f\inv (B))\) \subseteq \(B\text{;}\) if moreover \(f\) is surjective then \(f(f\inv(B)) = B\text{.}\)
