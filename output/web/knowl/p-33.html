For any positive integer \(n\text{,}\) let
\begin{equation*} \operatorname{GL}_n(\C)=\{ \text{invertible } n \times n \;\text{matrices with entries in}\; \C\}. \end{equation*}
By invertible I mean those matrices that have two-sided inverses, but in turns out that if an \(n \times n\) matrix has a left inverse then it it is automatically a right inverse too, and vice versa. Then \(\operatorname{GL}_n(\C)\) is a non-abelian group under matrix multiplication.
