Proposition 2.12. Criteria for Separability.

Let \(F\) be field and \(\ov{F}\) an [[Mathematics/Definitions/Algebraic Closure]] of \(F\text{.}\)
  1. Given \(f(x) \in F[x]\) and \(\a \in \ov{F}\text{,}\) the multiplicity of \(\a\) in \(f(x)\) is at least \(2\) if and only if \(f(\a) = 0\) and \(f'(\a) = 0\text{.}\)
  2. \(f\) is separable if and only if \(\gcd(f(x), f'(x)) = 1\) in \(F[x]\text{.}\)
  3. If \(f(x)\) is irreducible in \(F[x]\text{,}\) then \(f\) is separable if and only if \(f'(x) \neq 0\text{.}\)
