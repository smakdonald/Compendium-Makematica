Example 5.2. Common Group Actions.

  1. Trivial Action.
    For any group \(G\) and any set \(S\text{,}\) \(g \cdot s := s\) defines an action, the trivial action.
  2. Left Multiplication.
    For any group \(G\) and any set \(S\text{,}\) \(g \cdot s := gs\) defines an action, the left multiplication action 1 .
  3. Conjugation.
    For any group \(G\) and any set \(S\text{,}\) \(g \cdot s := gxg\inv\) defines an action, the conjugation action.
  4. The group \(S_n\) acts on the set \(X=\{1,2,\dots,n\}\) such that \(\sigma\cdot i=\sigma(i)\text{.}\)
  5. The group \(\GL_n(\R)\) acts on vectors in \(\R^n\) by left multiplication: given \(A\in\GL_n(\R)\) and \(\v\in\R^n\text{,}\) we have \(A\cdot\v=A\v\text{.}\)
