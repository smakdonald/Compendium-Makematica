Example 4.3. Common Group Actions.

  1. For any group \(G\) and any set \(S\text{,}\) \(g \cdot s := s\) defines an action, the trivial action. The associated group homomorphism is \(G \to \operatorname{Perm}(S)\) by \(g\mapsto \text{id}_S\text{.}\)
  2. The group \(S_n\) acts on the set \(X=\{1,2,\dots,n\}\) such that \(\sigma\cdot i=\sigma(i)\text{.}\)
  3. For any group \(G\) and any set \(S\text{,}\) \(g \cdot s := gs\) defines an action, the left multiplication action. The associated group homomorphism is \(G \to \operatorname{Perm}(S)\) by \(g\mapsto \text{id}_S\text{.}\)
  4. For any group \(G\) and any set \(S\text{,}\) \(g \cdot s := gxg\inv\) defines an action, the conjugation. The associated group homomorphism is \(G \to \operatorname{Perm}(S)\) by \(g\mapsto \text{id}_S\text{.}\)
  5. The group \(\GL_n(\R)\) acts on vectors in \(\R^n\) by left multiplication: given \(A\in\GL_n(\R)\) and \(\v\in\R^n\text{,}\) we have \(A\cdot\v=A\v\text{.}\)
