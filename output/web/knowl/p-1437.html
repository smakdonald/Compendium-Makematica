Consider the ring \(\R[x]\) and let \(I = (x^2 + 1)\) be the principal ideals generated by \(x^2+1\text{.}\) Since \(\R[x]\) is commutative, we have
\begin{equation*} (x^2 + 1) = \{ g(x)(x^2 + 1) \mid g(x)\in \R[x] \}, \end{equation*}
so \((x^2 + 1)\) is simply the collection of polynomials having \(x^2 + 1\) as a factor. I claim that \(\R[x]/(x^2 + 1)\) is isomorphic as a ring to \(\C\text{.}\) To prove this we define a map
\begin{equation*} \phi: \R[x] \to \C \end{equation*}
sending \(f(x)\) to \(f(i)\text{,}\) the evaluation of \(f\) at \(i\text{.}\) It is easy to check \(\phi\) is a ring homomorphism and it is onto since elements of the form \(a + bx\) in the source map to all possible complex numbers under \(\phi\text{.}\) We claim the kernel of \(\phi\) is \((x^2 +1)\text{.}\) It is clear that \(x^2 + 1 \in \ker(\phi)\) and it follows that \((x^2 + 1) \subseteq \ker(\phi),\) since \(\ker(\phi)\) is a two-sided ideal. Suppose \(\phi(f(x)) = 0\) and write \(f(x) = (x^2 + 1)q(x) + r(x)\) with degree of \(r(x)\) at most one, using the division algorithm in the polynomial ring \(\R[x]\text{.}\) So \(r(x) = a + bx\) for real numbers \(a,b\text{.}\) If \(r(x) \ne 0\text{,}\) then \(r(i) \ne 0\text{,}\) which would contradict \(f(i) = 0\text{.}\) So we must have \(r(x) = 0\) and hence \(f(x) \in (x^2 +1)\text{.}\) Applying the First Isomorphism Theorem for Rings, we get
\begin{equation*} \R[x]/(x^2+1) \cong \C \end{equation*}
via the map sending \(f(x) + (x^2 + 1)\) to \(f(i)\text{.}\) Intuitively, we have adjoined a formal symbol \(x\) to the real numbers, and by modding out \(x^2 + 1\) we have forced \(x\) to be a square root of \(-1\text{.}\) That is, we have adjoined \(i\) to the real numbers, obtaining \(\C\text{.}\)
