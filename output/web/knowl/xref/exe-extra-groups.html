Exercise 1.14. Some Extra Groups.

  • Opposite Group.
    Let \(G\) be a multiplicative group, and define \(G^{op}\) be the set \(G\) equipped with the operation \(*\text{,}\) where \(x*y=yx\text{.}\) Then \(G^{op}\) is a group.  6 
  • Evening the Odds.
    Let \(\E\) denote the set of even integers and \(\O\) denote the set of odd integers. Then \((\E,+)\) is a group, whereas \((\O,+)\) is not.
Solution.
  • Since \(G\) is a group and its operation is associative, we have \((y · x) · z = y · (x · z)\) for any elements \(x\text{,}\) \(y\text{,}\) \(z\) in \(G\text{.}\) This implies that \((z · y) · x = z · (y · x)\) for any elements \(x, y, z\) in \(G^{op}\text{.}\) Thus, associativity is satisfied in \(G^{op}\text{.}\)
    Let \(e\) be the identity element of \(G\text{.}\) In \(G^{op}\text{,}\) the identity element is still \(e\) because for any element \(x\) in \(G^{op}\text{,}\) we have \(e · x = x · e = x\text{,}\) where \(·\) denotes the operation in \(G\text{.}\) Therefore, the identity element property is satisfied in \(G^{op}\text{.}\)
    Let \(x\) be an element in \(G^{op}\text{.}\) Since \(x\) is also in \(G\text{,}\) \(G\) being a group implies that there exists an inverse element \(x\inv\) in \(G\) such that \(x · x\inv = x\inv · x = e\text{,}\) where \(e\) is the identity element of \(G\text{.}\) This implies that \(x\inv · x = x · x\inv = e\) in \(G^{op}\) as well. Therefore, every element in \(G^{op}\) has an inverse in \(G^{op}\text{,}\) satisfying the inverse element property.
  • Integer addition is associative, which means that for any three integers \(a, b\text{,}\) and \(c\text{,}\) the expression \((a + b) + c\) is equal to \(a + (b + c)\text{.}\) Thus, associativity is satisfied for \(\E\text{.}\)
    The identity element for addition is \(0\text{.}\) For any even integer \(a\text{,}\) adding \(0\) to it does not change its parity (evenness). Hence, the identity element property is satisfied for \(\E\text{.}\)
    For every even integer \(a\text{,}\) there exists an inverse element denoted as \(-a\) such that \(a + (-a) = (-a) + a = 0\text{.}\) The negative of an even integer is also an even integer. Adding an even integer to its negative results in \(0\text{,}\) which is the identity element. Therefore, the inverse element property is satisfied for \(\E\text{.}\)
    The identity element for addition is \(0\text{.}\) However, \(0\) is not an odd integer, so the identity element property is not satisfied for \(\O\text{.}\)
