Definition B.1. Torsion Subgroup.

Given an abelian group \(G\text{,}\) the set of elements of \(G\) with finite order forms a group, \(T\text{,}\) which we call the torision subgroup of \(G\text{.}\) If \(G=T\text{,}\) then we say that \(G\) is torsion
If the only element of \(G\) with finite order is the identity element \(e\text{,}\) we say that \(G\) is torision-free.
