First, suppose \(p<q\text{.}\) By Sylow's Theorem we know the following: - \(n_p|q\) and \(n_p\equiv 1\mod{p}\text{,}\) so \(n_p=1\) or \(q\text{,}\) so \(q\) - \(n_q|p^2\) and \(n_q\equiv 1\mod{q}\text{,}\) so \(n_q=1\) or \(p^2\text{,}\) so \(p^2\text{.}\) We know there must be \((q-1)\cdot p^2=p^2q-p^2\) elements of order \(p\text{.}\) Luckily, there is more than one Sylow \(p\)-subgroup with \(p^2\) elements, so there isn't room for all of them.
