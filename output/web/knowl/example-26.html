Example 3.5.

  1. A division ring \(D\) is both left Noetherian and left Artinian.
  2. If \(R\) is a PID but not a field (e.g., \(R=\mathbb{Z}\) or \(R=K[x]\)), then \(R\) is Noetherian but not Artinian. To see \(R\) is Noetherian, note that any ideal is of the form \(I=(p_1^{e_1} \cdots p_t^{e_t})\) for some irreducible elements \(p_i\) and positive integers \(e_i\text{.}\) An ideal contains \(I\) if it corresponds to a product of the same irreducibles with smaller or equal multiplicities; there are only finitely many of these so an ascending chain must stablilize. To see \(R\) is not Artinian, take some irreducible \(p\) and take the chain
    \begin{equation*} (p) \supsetneqq (p^2) \supsetneqq (p^3) \supsetneqq (p^4) \supsetneqq \cdots . \end{equation*}
  3. A polynomial ring in infinitely many variables is neither Noetherian nor Artinian: there is an ascending chain
    \begin{equation*} (x_1) \subsetneqq (x_1,x_2) \subsetneqq (x_1,x_2,x_3) \subsetneqq (x_1,x_2,x_3,x_4) \subsetneqq \cdots \end{equation*}
    and take a descending chain as in the last example.
  4. The \(\mathbb{Z}\)-module \(M=\displaystyle \frac{\mathbb{Z}[\frac12]}{\mathbb{Z}}\text{,}\) where \(\mathbb{Z}[\frac12]\) is the subring of \(\mathbb{Q}\) generated by \(\mathbb{Z}\) and \(\frac12\text{,}\) is Artinian but not Noetherian. Suppose that \(N\subseteq M\) is generated by \(\displaystyle \big\{ \big[\frac{a_\lambda}{2^{n_\lambda}}\big]\big\}\text{,}\) where each \(a_\lambda\) is odd (we can write any element in \(\mathbb{Z}[\frac12]\) like so). Observe that for each \(\lambda\text{,}\) there are integers \(s,t\) such that \(\displaystyle s a_\lambda + t 2^{n_\lambda} = 1\text{,}\) so \(\displaystyle s \big[\frac{a_\lambda}{2^{n_\lambda}}\big] = \big[\frac{1}{2^{n_\lambda}}\big]\text{.}\) Thus, \(N\) is generated by \(\displaystyle \big\{ \big[\frac{1}{2^{n_\lambda}}\big]\big\}\text{.}\) Thus, the submodules of \(M\) are \(M\) itself, \(0\text{,}\) and \(M_i = \displaystyle \frac{\mathbb{Z}\cdot \frac{1}{2^n}}{\mathbb{Z}}\) for \(i>0\text{.}\) We have \(0 \subsetneqq M_1 \subsetneqq M_2 \subsetneqq \cdots\) so \(M\) is not Noetherian. However, any descending chain is either always equal to \(M\text{,}\) or else has some \(M_i\) as a term, and there are finitely many submodules of such an \(M_i\text{,}\) so must stabilize.
  5. The subring of \(M_2(\mathbb{Q})\) given as
    \begin{equation*} \left\{ \begin{bmatrix} a & 0 \\ b & c \end{bmatrix} \ \big| \ a\in \mathbb{Z}, b,c\in \mathbb{Q}\right\} \end{equation*}
    is left Noetherian but not right Noetherian.
