Problem 4.26.

Let \(R\) be a commutative ring with \(1\text{.}\) Recall that \(r \in R\) is called nilpotent if \(r^n = 0\) for some integer \(n > 0.\) Let \(N\) be the set of nilpotent elements of \(R\text{.}\)
  1. Show that \(N\) is an ideal of \(R\text{.}\)
  2. Show that the ring \(R/N\) has no nonzero nilpotent elements.
  3. For a polynomial \(f(x) \in R[x]\text{,}\) prove that \(f(x)\) is a nilpotent element of \(R[x]\) if and only if every coefficient of \(f(x)\) is nilpotent.
