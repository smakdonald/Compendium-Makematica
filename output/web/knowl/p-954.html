  1. \(f\) is a unit in \(A[x]\) if and only if \(a_{0}\) is a unit in \(A\) and \(\adots\) are nilpotent.
  2. \(f\) is nilpotent if and only if \(a_{i}\) nilpotent for all \(i\text{.}\)
  3. \(f\) is a zerodivisor if and only if there exists \(a\neq0\) in \(A\) such that \(af=0\text{.}\)
  4. \(f\) is said to be primitive if \((a_{0},\adots)=(1)\text{.}\) Prove that if \(f,g\in A[x]\text{,}\) then \(fg\) is primitive if and only if both \(f\) and \(g\) are primitive.
