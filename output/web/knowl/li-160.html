Item 4.

The \(\mathbb{Z}\)-module \(M=\displaystyle \frac{\mathbb{Z}[\frac12]}{\mathbb{Z}}\text{,}\) where \(\mathbb{Z}[\frac12]\) is the subring of \(\mathbb{Q}\) generated by \(\mathbb{Z}\) and \(\frac12\text{,}\) is Artinian but not Noetherian. Suppose that \(N\subseteq M\) is generated by \(\displaystyle \big\{ \big[\frac{a_\lambda}{2^{n_\lambda}}\big]\big\}\text{,}\) where each \(a_\lambda\) is odd (we can write any element in \(\mathbb{Z}[\frac12]\) like so). Observe that for each \(\lambda\text{,}\) there are integers \(s,t\) such that \(\displaystyle s a_\lambda + t 2^{n_\lambda} = 1\text{,}\) so \(\displaystyle s \big[\frac{a_\lambda}{2^{n_\lambda}}\big] = \big[\frac{1}{2^{n_\lambda}}\big]\text{.}\) Thus, \(N\) is generated by \(\displaystyle \big\{ \big[\frac{1}{2^{n_\lambda}}\big]\big\}\text{.}\) Thus, the submodules of \(M\) are \(M\) itself, \(0\text{,}\) and \(M_i = \displaystyle \frac{\mathbb{Z}\cdot \frac{1}{2^n}}{\mathbb{Z}}\) for \(i>0\text{.}\) We have \(0 \subsetneqq M_1 \subsetneqq M_2 \subsetneqq \cdots\) so \(M\) is not Noetherian. However, any descending chain is either always equal to \(M\text{,}\) or else has some \(M_i\) as a term, and there are finitely many submodules of such an \(M_i\text{,}\) so must stabilize.
