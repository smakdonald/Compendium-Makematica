Next, note that \(\sqrt[p]{2}\) is a root of the polynomial \(x^p-2\text{,}\) making it algebraic over \(\Q\text{.}\) Using Eisenstein's Criterion in conjunction with the prime \(q=2\) we see that \(x^p-2\) is irreducible in \(\Q[x]\text{.}\) As \(\gcd(p,p-1)=1\text{,}\) by Problem 5 on the previous homework we see that \(x^p-2\) is irreducible in \(\Q(\sqrt[p]{2})\) as well. As \(x^p-2\) is irreducible and monic, we see that it is the minimal polynomial of \(\sqrt[p]{2}\) in \(\Q(e^{2\pi i/p})[x]\text{,}\) and thus
\begin{equation*} [\Q(e^{2\pi i/p},\sqrt[p]{2}):\Q(e^{2\pi i/p})]=p. \end{equation*}
Using the degree formula, we see that
\begin{equation*} [\Q(\sqrt[p]{2},e^{2\pi i/p}):\Q]=[\Q(e^{2\pi i/p},\sqrt[p]{2}):\Q(e^{2\pi i/p})][\Q(\sqrt[p]{2}):\Q]=p(p-1). \end{equation*}
Notice that \(\sqrt[p]{2}\cdot (e^{{2\pi i/p}^n})\text{,}\) where \(1\leq n\leq p\text{,}\) which are all roots of \(x^p-2\text{.}\) By the Factor Theorem, there can be no other roots of this polynomial. Thus \(L\subseteq\Q(\sqrt[p]{2},e^{2\pi i/p})\text{.}\)
