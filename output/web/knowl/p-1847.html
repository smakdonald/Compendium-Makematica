Notice that \(\a_n\sqrt[n]2\in\overline{\Q}\) for all \(n\in\N\) and that \(\a_n\) is a root of the polynomial \(x^n-2\text{,}\) which is irreducible in \(\Q[x]\) by Eisenstein's Criterion (\(p=2\)). Thus \([\Q(\a_n):\Q]=n\text{.}\) As \(\a_n\) must be added to \(\Q\) for all \(n\text{,}\) we see that this extension is not finite.
