Theorem 1.7. Properties of Groups.

Let \(G\) be a group.
  1. Unique Identity.
    The element \(e\in G\) satisfying Item 2 of Definition 1.2 is unique, and we thus refer to it as the identity element of \(G\text{.}\)
  2. Unique Inverses.
    For each \(x\in G\text{,}\) the element satisfying Item 3 of Definition 1.2 is unique, and we thus refer to it as the inverse of \(x\text{.}\)
  3. Cancellation.
    Suppose \(x,y,z\in G\text{.}\) If \(xy = xz\) then \(y=z\text{.}\) Similarly, if \(yx = zx\) we have \(y = z\)
