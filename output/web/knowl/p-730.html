If \(f: R \to S\) is a ring homomorphism and \(I \subseteq R\) is an ideal such that \(I \subseteq \ker(f)\)[^1], there exists a well defined ring homomorphism \(\overline{f}: R/I \to S\) such that \(\overline{f} (r+I) = f(r)\text{.}\) Furthermore, if \(f\) is surjective then \(\overline{f}\) is surjective and if \(I=\ker(f)\) then \(\overline{f}\) is injective.
