Throughout this course we will encounter numerous important subgroups that will need to be verified as such. We do so here. While these subgroups may seem random now, they will prove to be exceedingly useful in the future.
  1. Let \(H\leq G\text{.}\) Prove that the set \(Z_G(H)=\{g\in G|gh=xh\forall h\in H\}\) is a subgroup of \(G\text{.}\)  3 
  2. Let \(H\leq G\text{.}\) Prove that the set \(N_G(H)=\{g\in G|gHg\inv=H\}\) is a subgroup of \(G\text{.}\)  4 
  3. Let \(H\) be a subgroup of \(G\) and fix some element \(g\in G\text{.}\) Prove that \(gHg^{-1}=\{ghg^{-1} \mid h\in H\}\) is a subgroup of \(G\) of the same order as \(H\text{.}\)  5 
  4. Let \(S\) be a containing \(n\) elements. For each \(x\in S\text{,}\) prove that the set \(\Stab_{\Perm(S)}(x)=\{\sigma\in \Perm(S)|\sigma(x)=x\}\) is a subgroup of \(\Perm(S)\text{.}\)  6
