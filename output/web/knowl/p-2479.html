Let \(A\in\GL_3(\F_2)\text{.}\) All characteristic polynomials are monic, and as \(A\) is invertible we see the \(a_0\) term in the \(\cp_A(x)\neq0\text{,}\) and thus \(a_0=1\text{,}\) the only other element in \(\F_2\text{.}\) There are only so many monic polynomials with coefficients in \(\F_2\text{;}\) hence there are only four possible characteristic polynomials of \(A\text{:}\) 1. \(f(x)=x^3+x^2+x+1\text{,}\) 2. \(g(x)=x^3+x^2+1\text{,}\) 3. \(h(x)=x^3+x+1,\) and 4. \(j(x)=x^3+1\text{.}\) Note that since \(a_0=1\text{,}\) 0 cannot be a root of any of these polynomials. Thus all that remains is to check \(1\text{.}\) Luckily, \(1\) is not a root of \(g(x)\) or \(h(x)\text{,}\) so the only invariant factor of each is themselves. While 1 is a root of \(j(x)\text{,}\) we see that \(j(x)=(x+1)(x^2+x+1)\text{.}\) As \((x+1)\not|(x^2+x+1)\text{,}\) neither of these polynomials can be invariant factors by the RCF theorem; thus the only invariant factor of \(j(x)\) is itself. However \(f(x)=(x+1)^3\text{,}\) and so its possible invariant factors are - \((x+1)^3\text{;}\) - \((x+1),(x+1)^2;\) and - \((x+1),(x+1),(x+1)\text{.}\) Let \(k(x)=(x+1)^2\) and \(\ell(x)=x+1\text{.}\) Thus the conjugacy class representatives for \(\GL_3(\F_2)\) are 1. \(C(f)\text{,}\) 2. \(C(\ell)\oplus C(k)\text{,}\) 3. \(C(\ell)\oplus C(\ell)\oplus C(\ell)\text{,}\) 4. \(C(g)\text{,}\) 5. \(C(h)\text{,}\) and 6. \(C(j)\text{.}\)
