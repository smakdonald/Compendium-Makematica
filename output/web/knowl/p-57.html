Let \(G\) be a group.
  1. Unique Identity.
    The element \(e\in G\) satisfying the Identity Property of Definition 1.3 is unique, and we thus refer to it as the identity element of \(G\text{.}\)
  2. Unique Inverses.
    For each \(x\in G\text{,}\) the element satisfying the Inverse Property of Definition 1.3 is unique, and we thus refer to it as the inverse of \(x\text{.}\)
  3. Cancellation.
    Suppose \(x,y,z\in G\text{.}\) If \(xy = xz\) then \(y=z\text{.}\) Similarly, if \(yx = zx\) we have \(y = z\text{.}\)
