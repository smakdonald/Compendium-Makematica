Theorem A.13. Preimages are Nice.

If f: X \to Y, E \subseteq E' \subseteq Y, and E\alpha \subseteq Y for all \alpha , then
  1. f -1(E) \subseteq f -1(E').
  2. f -1(\cup \alpha E\alpha ) = \cup \alpha f -1(E\alpha ).
  3. f -1(\cap \alpha E\alpha ) = \cap \alpha f -1(E\alpha ).
  4. f -1(E' - E) = f -1(E') - f -1(E).
