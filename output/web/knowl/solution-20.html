Solution 3.1.
Let \(G\) act on the left cosets of \(K\) in \(G\) by left multiplication, yielding the permutation representation homomorphism \(\rho:G\to S_n\text{.}\) Let \(N\) conspicuously denote the kernel of this function.
Let \(x\in N\text{.}\) Then \(\rho(n)\text{,}\) the automorphism \(\s:G/K\to G/K\) defined by \(\s(gK)=xgK\) is precisely the identity permutation. Thus \(xgK=gK\) for all \(g\in G\) and \(g\inv xgK=K\) for all \(g\in G\text{,}\) so \(x\in\) This means the elements of \(N\) are precisely those that are in the conjugacy class of \(gKg\inv\) for all \(g\in G\text{.}\) There cannot exist a larger normal subgroup of \(G\) contained in \(K\text{,}\) as it would contain an element that was not in some conjugacy class for \(G\text{,}\) negating the definition of a normal subgroup.
Recall the permutation representation homomorphism \(\rho:G\to S_n\text{.}\) The First Isomorphism Theorem tells us \(G/N\cong\im(\rho)\leq S_n\text{,}\) which has order \(n!\text{.}\) Thus \([G:N]|n!\text{.}\)
