    Section 1.1 Group Basics

    Subsection Binary Operations and Algebraic Structures

    “The operation was a success, but I'm afraid the doctor is dead.”
    ―Steve Martin

    A Brief Notational Aside.

    At various points throughout this text, the author may use "mathcal" notation for definitions that do not normally require it (such as \(\G\) for a group, \(\R\) for a ring, etc.). In the realm of higher mathematics, this is what is known as a typo, and should be reported to the proper authorities as soon as possible.
    We begin, as all great volumes of mathematics are like to do, somewhere in the middle.  1 

    Definition 1.1. Binary Operation.

    A binary operation on a set \(S\) is a Function
    \begin{equation*} -*-:S\times S\to S,\text{ given by } (x,y)\mapsto x * y. \end{equation*}

    Example 1.2. Binary Operations.

    • Addition (\(-+-\)) and multiplication (\(-\cdot-\)) are both binary operations on \(\Z,\Q,\R,\) and \(\C\text{.}\)
    • Let \(\cP(S)\) denote the power set of a set \(S\text{.}\) Then
      \begin{equation*} -\cup-:\cP(S)\to \cP(S)\text{ given by } (X,Y)\mapsto X\cup Y \end{equation*}
      and
      \begin{equation*} -\cap-:\cP(S)\to \cP(S)\text{ given by } (X,Y)\mapsto X\cap Y \end{equation*}
      are both binary operations on \(\cP(S)\text{.}\)
    • Let \(F(A)\) denote the set of all functions from \(S\) to itself. Then
      \begin{equation*} -\circ-:F(A)\to F(A)\text{ given by } (f,g)\mapsto f\circ g \end{equation*}
      is a binary operation on \(F(A)\text{.}\)

    Definition 1.3. Binary Properties.

    Let \(-*-\) be a binary operation on a set \(S\text{.}\)
    1. An operation satisfies the associative property if for all \(x,y,z\in S\) we have \((x*y)*z=x*(y*z)\text{.}\)
    2. An operation satisfies the commutative property if \(x * y = y * x\) for all \(x,y \in S\text{.}\)
    3. An operation satisfies the identity property if there exists \(e\in G\) such that \(e*g=x*g=x\) for all \(g\in G\text{.}\) Such an element \(e\) is called an identity element.
    4. An operation satisfies the inverse property if for each \(x\in S\text{,}\) there is an element \(y\in S\) such that \(x * y=e=y * x\text{,}\) where \(e\) is an identity element of \(S\text{.}\) Such an element \(y\) is called an inverse of the element \(x\text{.}\)

    Convention 1.4.

    The notation \(*\) is used for an arbitary binary operation. When context is sufficient for understanding, we often write \(xy\) instead of \(x * y\) for simplicity.
    The notation \(\cdot\) is also sometimes used for an arbitary binary operation, though we avoid doing so here to prevent confusion with the mutliplication operation, as seen in in Example 1.2.

    Definition 1.5. Algebraic Structures.

    • A magma (or binary structure) is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\text{.}\)
    • A semigroup is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\) satisfying the associative property.
    • A monoid is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\) satisfying the associative and identity properties.
    • A group is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\) satisfying the associative, identity, and inverse properties.
    • An abelian group is a pair \((G, *)\) where \(G\) is a set and \(*\) is a binary operation on \(G\) satisfying the associative, identity, inverse, and commutative properties.

    Convention 1.6.

    Though a group is a pair, we will usually refer to the group by only naming the underlying set, \(G\text{.}\)

    Mark 1.7.

    Magmas, semigroups, and monoids won't show up all that much moving forward, but they are doing a surprising amount of leg work under the surface, and it's worth mentioning them. However, if you're feeling overwhelmed by the amount of new words buzzing around, those three are probably the ones you can worry about the least, at least in this text.  2

    Subsection Groups

    “Most great learning happens in groups.”
    ―Ken Robinson

    Theorem 1.9. Properties of Groups.

    Let \(G\) be a group.
    1. Unique Identity.
      The element \(e\in G\) satisfying Item 3 of Definition 1.3 is unique, and we thus refer to it as the identity element of \(G\text{.}\)
    2. Unique Inverses.
      For each \(x\in G\text{,}\) the element satisfying Item 4 of Definition 1.3 is unique, and we thus refer to it as the inverse of \(x\text{.}\)
    3. Cancellation.
      Suppose \(x,y,z\in G\text{.}\) If \(xy = xz\) then \(y=z\text{.}\) Similarly, if \(yx = zx\) we have \(y = z\text{.}\)

    Proof.

    1. Suppose there exist two elements \(e\) and \(e'\) such that \(ex = x = xe\) and \(e'x = x = xe'\) for all \(x\text{.}\) Then
      \begin{equation*} e = ee' = e'. \end{equation*}
      Thus \(e=e'\text{,}\) making the identity element unique.
    2. For a given \(x\text{,}\) if \(yx = xy = e\) and \(zx = xz = e\) for some \(y\) and \(z\text{,}\) then
      \begin{equation*} \begin{aligned} z &= ez \\ &= (yx)z \\ &= y(xz) \\ &= ye \\ &= y. \end{aligned} \end{equation*}
      Thus \(y=z\text{,}\) making inverses unique.
    3. Suppose \(xy = xz\) for some \(x,y,z\in G\text{.}\) By the Item 4, there exists some \(x\inv\in G\) such that \(xx\inv=e=x\inv x\text{.}\) Observe
      \begin{equation*} \begin{aligned} xy &= xz \\ x\inv(xy) &= x\inv(xz) & \text{multiplying on the left by }x\inv \\ (x\inv x)y &= (x\inv x)z & \text{associativity} \\ (e)y &= (e)z & \text{inverses}\\ y &= z & \text{identity}. \end{aligned} \end{equation*}

    Remark 1.10.

    The Unique Identity property of Theorem 1.9 holds in monoids (and unital magmas) as well.

    Convention 1.11.

    The inverse of an element \(x\) is often written \(x\inv\text{,}\) for reasons we will see in the next section.

    Example 1.12. Group Examples.

    1. The trivial group is the group with a single element \(\{e\}\text{.}\)
    2. \((\Z,+), (\Q,+), (\R,+),\) and \((\C,+)\) each form an abelian group, where \(+\) denotes traditional addition. The axioms of arithmetic guarantee the validity of the group axioms as well as the commutativity of the group operation. Thus all four groups are abelian.
    3. For any positive integer \(n\text{,}\) let
      \begin{equation*} \operatorname{GL}_n(\C)=\{ \text{invertible } n \times n \;\text{matrices with entries in}\; \C\}. \end{equation*}
      Then \(\operatorname{GL}_n(\C)\) is a non-abelian group under matrix multiplication known as the general linear group.
    4. For any \(n\in\N\text{,}\) let \(\Z/n\) denote the The Integers Modulo \(n\). Then \((\Z/n,+)\) forms an abelian group where \(+\) denotes addition modulo \(n\text{.}\)

    Identity Theft.

    In familiar groups of numbers, “\(1\)” is used for the identity element of a multiplicative group (and likewise “\(0\)” in additive groups). Often, “\(1\)” is used for the identity element \(e\) in arbitrary groups, though we will not do so here.

    Remark 1.13. Specific General Linear Groups.

    In general, \(\GL_n(F)\) is a group for any field \(F\) (whatever those are).  3 
    The following example illustrates a few important examples of something called a Group of Units, which is defined explicitely in terms of rings (whatever those are) in the aptly named Ring Theory.

    Example 1.14. Groups of Units.

    Each of the following subsets form an abelian group under multiplication.
    1. The subset \(\Z^\times=\{-1,1\}\) of \(\Z\text{.}\)
    2. \(\Q^\times=\Q\setminus\{0\}\text{,}\) \(\R^\times=\R\setminus\{0\}\text{,}\) and \(\C^\times=\C\setminus\{0\}\text{.}\)
    3. For each \(n\in\N\text{,}\) the subset \(\Z/n^\times=\{k\in\Z/n|\gcd(k,n)=1\}\) of \(\Z/n\text{.}\)
    Some texts use the notation “\(^*\)” to denote groups of units. For example, \(\Q^\times\) would be written \(\Q^*\text{.}\)

    Exploration 1.15. \(\Z/p^\times\).

    Prove that \(\Z/p^\times=\mathbb{Z}/p\setminus\{0\}\text{.}\)

    Exploration 1.16. Some Extra Groups.

    • Opposite Group.
      Let \(G\) be a multiplicative group, and define \(G^{op}\) be the set \(G\) equipped with the operation \(*\text{,}\) where \(x*y=yx\text{.}\) Then \(G^{op}\) is a group.  4 
    • Evening the Odds.
      Let \(\E\) denote the set of even integers and \(\O\) denote the set of odd integers. Then \((\E,+)\) is a group, whereas \((\O,+)\) is not.
    Solution.
    • Since \(G\) is a group and its operation is associative, we have \((y · x) · z = y · (x · z)\) for any elements \(x\text{,}\) \(y\text{,}\) \(z\) in \(G\text{.}\) This implies that \((z · y) · x = z · (y · x)\) for any elements \(x, y, z\) in \(G^{op}\text{.}\) Thus, associativity is satisfied in \(G^{op}\text{.}\) Let \(e\) be the identity element of \(G\text{.}\) In \(G^{op}\text{,}\) the identity element is still \(e\) because for any element \(x\) in \(G^{op}\text{,}\) we have \(e · x = x · e = x\text{,}\) where \(·\) denotes the operation in \(G\text{.}\) Therefore, the identity element property is satisfied in \(G^{op}\text{.}\) Let \(x\) be an element in \(G^{op}\text{.}\) Since \(x\) is also in \(G\text{,}\) \(G\) being a group implies that there exists an inverse element \(x\inv\) in \(G\) such that \(x · x\inv = x\inv · x = e\text{,}\) where \(e\) is the identity element of \(G\text{.}\) This implies that \(x\inv · x = x · x\inv = e\) in \(G^{op}\) as well. Therefore, every element in \(G^{op}\) has an inverse in \(G^{op}\text{,}\) satisfying the inverse element property.
    • Integer addition is associative, which means that for any three integers \(a, b\text{,}\) and \(c\text{,}\) the expression \((a + b) + c\) is equal to \(a + (b + c)\text{.}\) Thus, associativity is satisfied for \(\E\text{.}\) The identity element for addition is \(0\text{.}\) For any even integer \(a\text{,}\) adding \(0\) to it does not change its parity (evenness). Hence, the identity element property is satisfied for \(\E\text{.}\) For every even integer \(a\text{,}\) there exists an inverse element denoted as \(-a\) such that \(a + (-a) = (-a) + a = 0\text{.}\) The negative of an even integer is also an even integer. Adding an even integer to its negative results in \(0\text{,}\) which is the identity element. Therefore, the inverse element property is satisfied for \(\E\text{.}\) The identity element for addition is \(0\text{.}\) However, \(0\) is not an odd integer, so the identity element property is not satisfied for \(\O\text{.}\)

    Theorem 1.17. Properties of Inverses.

    If \(G\) is a group and \(x,y\in G\text{,}\) then
    1. \((x^{-1})^{-1} = x\) and
    2. \(\displaystyle (xy)\inv=y\inv x\inv\)

    Proof.

    1. We show that both \(x\) and \((x^{-1})^{-1}\) are inverses of the element \(x\inv\text{.}\) First, recall from Item  that \(x\cdot x\inv=e\text{.}\) However, it is also the case that \((x\inv)\inv\cdot x\inv=e\text{.}\) By part (2) of Theorem 1.9 we know that inverses are unique, and thus it must be the case that \((x^{-1})^{-1} = x\text{.}\)
    2. By definition, \((xy)\inv\) is the inverse of the element \(xy\text{.}\) However, we also have
      \begin{equation*} \begin{aligned} (xy)(y\inv x\inv) &= x(yy\inv)x\inv \\ &= x(e)x\inv\\ &= (xe)x\inv \\ &= xx\inv \\ &= e. \end{aligned} \end{equation*}
      Thus both \((xy)\inv\) and \(y\inv x\inv\) are inverses of \((xy)\text{,}\) making them equal by part (2) of Theorem 1.9.
    The second portion of Theorem 1.17 can be generalized.

    Exploration 1.18. Big 'Ole Inverses.

    If \(G\) is a group and \(x_1,\ldots,x_n \in G\text{,}\) then \((x_1 \dots x_n)^{-1} = x_n^{-1} \dots x_1^{-1}\text{.}\)

    Exploration 1.19. Generalized Associative Law.

    If an element of a monoid is contructed from a sequence of elements \(x_1,\dots,x_n\) in this order by repeatedly inserting parenthesis and applying the operation, the element must equal
    \begin{equation*} (\cdots((x_1x_2)x_3)\cdots)x_n \end{equation*}
    and so is independent of the mode of bracketing.  5 
    Solution.
    Certainly we may assume that \(n > 2\text{.}\) If \(u\) is an element constructed from \(x_1, x_2 ,..., x_n\) in the prescribed manner, we can write \(u = vw\) where \(v\) and \(w\) are constructed from \(x_1 ,x_2 ,...,x_f\) and \(x_m , . . . , x_n\) respectively \((1 < i < n)\text{.}\) If \(w = x_n\) , the result follows by induction on \(n\text{.}\) Otherwise we can write \(w = w'(x_n)\) and \(u = (vw')x_n\) : once again the result follows by induction on n.
    As a result of Exploration 1.19 any expression formed from the elements \(x_1\dots,x_n\) in that order can be written without parenthesis, which will save us oodles of time down the line.  6
