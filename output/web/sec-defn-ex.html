    Section 1.1 Intro to Rings

    Subsection Definition and First Properties

    Definition 1.1.

    A ring is a set \(R\) equipped with two binary operations, \(+\) and \(\cdot\text{,}\) satisfying:
    1. \((R,+)\) is an abelian group with identity element denoted \(0\text{,}\)
    2. \(\cdot\) is associative (making \((R,\cdot)\) a semigroup)
    3. \(a \cdot (b + c) = a \cdot b + a \cdot c\) and \((a + b) \cdot c = a \cdot c + b \cdot c\) hold for all \(a,b,c \in R\text{.}\)
    \(R\) is a unital ring (or a ring with identity) if, in addition to (1), (2), (3)
    1. there is a multiplicative identity element written as \(1\) such that \(1 \cdot a = a = a \cdot 1\) for all \(a \in R\text{.}\)
    \(R\) is commutative if in addition to (1)-(3)
    1. \(a \cdot b = b \cdot a\) holds for all \(a,b \in R\text{.}\)
    \(R\) is a division ring if \(1\neq 0\text{,}\) (1)–(4) and (6) hold
    1. \((R - \{0\})\) is a group under \(\cdot\) (i.e. every \(r\in (R - \{0\})\) has a multiplicative inverse)
    \(R\) is a field if \(1\neq 0\) and (1)–(6) hold (i.e. a field is a commutative division ring).

    Problem 1.2.

    Commutativity of addition is a consequence of the other ring axioms in rings with \(1\text{.}\)
    Here are some basic consequences of the axioms.

    Proposition 1.3. Ring Arithmetic.

    For any ring \(R\) and all \(a,b \in R\) we have:
    1. \(a \cdot 0 = 0 = 0 \cdot a\text{,}\)
    2. \((-a)b = -(ab) = a(-b)\text{,}\)
    3. \((-a)(-b) = ab\text{.}\)
    If moreover \(R\) is unital, then
    1. \(1\) is unique, and
    2. \((-1)a = -a\text{.}\)

    Proof.

    Let \(R\) be a ring with identity.
    1. Let \(a\in R\text{.}\) Observe that \(0a=(0+0)a=0a+0a\) by the distributive law. Subtracting \(0a\) from both sides we have \(0=0a\text{.}\) A similar argument shows that \(0=a0\text{.}\)
    2. Let \(a\in R\text{.}\) Note that \(-a+a=0\text{.}\) Consider \((-1)a+a\text{.}\) Using the distributive law we see that \((-1)a+a=a(-1+1)=a(0)=0\text{.}\) Thus \(-a\) and \((-1)a\) are both additive inverses of \(a,\) making them unique.

    Example 1.4.

    1. \(R=\{0\}\) is called the trivial ring. Notice that in the trivial ring \(0=1\text{.}\) Conversely, if \(1 = 0\) in a ring, then \(R = \{0\}\text{,}\) since in this case for all \(a\text{,}\) we have \(a \cdot 0 = 0\) and hence \(a = a \cdot 1 = a \cdot 0 = 0\text{.}\)
    2. \(\mathbb{Z}\) is a commutative ring.
    3. \(\mathbb{Z}/n\) is a commutative ring under addition and multiplication modulo \(n\text{.}\) Note that \(\mathbb{Z}/n\) is a field if any only if \(n\) is prime.
    4. The familiar sets of “numbers" \(\mathbb{Q}, {\mathbb{R}}, \mathbb{C}\) are fields.
    5. (Matrix ring) If \(R\) is any ring (not necessarily commutative), so is \(\operatorname{M}_{n}(R)\) for any natural number \(n\text{,}\) using the usual rules for addition and multiplication of square matrices.
    6. (The real Hamiltonian quaternion ring) Let \(i,j,k\) be formal symbols and set \({\mathbb{H}}\) to be the four dimensional \({\mathbb{R}}\)-vector space consisting of all expressions of the form \(a + bi + cj + dk\) with \(a,b,c,d \in {\mathbb{R}}\text{.}\) Addition is vector space addition:
      \begin{equation*} (a + bi + cj + dk) +(a' + b'i + c'j + d'k) = (a + a') + (b + b') i + (c + c')j + (d + d')k. \end{equation*}
      Multiplication is uniquely determined by the axioms of a ring together with the rules
      \begin{equation*} i^2 = j^2 = k^2 = -1, -ji = ij = k, -kj = jk = i, -ik = ki = j. \end{equation*}
      and the fact that the real coefficients commute with each other and \(i,j,k\text{.}\)
      It's not obvious that the multiplication defined in this way satisfies associativity, but in fact it does (this amounts conditions very similar to the associativity of the group \(Q_8\)).
      \({\mathbb{H}}\) is a division ring, since one can check that
      \begin{equation*} (a + bi + cj + dk)^{-1} = \frac{a - bi - cj - dk}{\|a + bi + cj + dk\|} \end{equation*}
      where
      \begin{equation*} \|a + bi + cj + dk \| := a^2 + b^2 + c^2 + d^2. \end{equation*}
      In the equation above \(\|a + bi + cj + dk \|\) is non-zero real number if \(a + bi + cj + dk\) is not the zero element. The quantity \(\|a + bi + cj + dk \|\) is called the norm of the quaternion \(a + bi + cj + dk\text{.}\)
    7. (Direct product of rings) The cartesian product \(R\times R'\) of two rings \(R\) and \(R'\) has a natural ring structure with addition and multiplication defined componentwise:
      \begin{equation*} (a,b)+(c,d)=(a+c,b+d) \end{equation*}
      \begin{equation*} (a,b)\cdot(c,d)=(a\cdot c,b\cdot d) \end{equation*}
    8. If \(X\) is a set and \(R\) is a ring, let \(\operatorname{Fun}(X, R)\) be the collection of set theoretic functions from \(X\) to \(R\text{,}\) and define \((f + g)(x) = f(x) + g(x)\) and \((f \cdot g)(x) := f(x) \cdot g(x)\text{.}\) Then \(\operatorname{Fun}(X,R)\) is a ring. If \(X\) is a finite set and \(|X|=n\text{,}\) then \(\operatorname{Fun}(X,R)\) may be identified with \(R^n=\underbrace{R \times \cdots \times R}_n\text{,}\) the direct product of \(n\) copies of \(R\text{.}\)
    9. (Endomorphism ring) If \(A = (A, +)\) is any abelian group, set \(\operatorname{End}_{Ab}(A)\) to be the collection of endomorphisms of \(A\) — that is, the set of group homomorphisms \(f: A \to A\) from \(A\) to itself.
      Then \(\operatorname{End}_{Ab}(A)\) is a ring with addition \((f + g)(a) := f(a) + g(a)\) and multiplication \(f \cdot g := f \circ g\text{.}\) This is almost always a non-commutative ring.

    Subsection Units, Zero Divisors, Integral Domains

    Definition 1.5.

    An element \(a\) of a unital ring \(R\) with \(1\neq 0\) is called a unit there exists \(b\in R\) such that\(ab=1\) and \(ba=1\text{.}\) In case such \(b\) exists, it is unique, it is called the inverse of \(a\) and denoted by \(a^{-1}\text{.}\)

    Definition 1.6.

    The set of units of a non-trivial unital ring \(R\) is denoted \(R^\times\text{.}\) This forms a group \((R^\times,\cdot)\) with respect to multiplication.

    Problem 1.7.

    The group of units is indeed a group \((R^\times,\cdot)\) with respect to multiplication.

    Example 1.8.

    \(\operatorname{M}_{n}(F)^\times = \operatorname{GL}_n(F)\text{.}\)

    Problem 1.9.

    Let \(F\) be a field. Prove that every non-zero element of \(F\) is a unit. In symbols, show \(F^\times=F\setminus\{0\}\text{.}\)

    Definition 1.10.

    A zero-divisor in a ring \(R\) is an element \(x\ne 0\) such that \(xy = 0\) or \(yx=0\) for some \(y \ne 0\text{.}\)

    Definition 1.11.

    A unital ring \(R\) is an integral domain (often shortened to domain) if \(1 \ne 0\text{,}\) \(R\) is commutative, and \(R\) has no zero divisors.

    Lemma 1.12.

    Let \(R\) be a commutative ring with identity. Then \(R\) is an integral domain if and only if for all \(x,y,z\in R\) such that \(xy=xz\text{,}\) we have \(y=z\text{.}\) This property is called cancellation

    Example 1.13.

    1. Every field is an integral domain.
    2. \(\Z/n\) is an integral domain if and only if \(n\) is prime (in which case it happens to be a field too) or \(n = 0\) (in which case \(\Z/0\cong \Z\)).

    Problem 1.14.

    Prove that a finite integral domain \(D\) must be a field.
    Proof.
    Let \(x\in D\text{,}\) and consider the set \(\{x^n|n\in\N\}\text{.}\) As \(D\) is finite there must exist \(n<m\in\N\) such that \(x^n=x^m\text{.}\) In integral domains cancellation is allowed, and thus we can cancel an \(x^n\) from both sides, yielding \(e=x^{m-n}\text{,}\) and thus \(x\cdot x^{m-n-1}=e\text{,}\) making \(x\) a unit of \(D\text{.}\) Thus \(D\) is a field.

    Lemma 1.15.

    If \(a\) is a zero divisor in a ring \(R\text{,}\) then \(a\) is not a unit.

    Proof.

    Suppose that \(a\) is both a zero divisor and a unit. Then there exists \(b\neq 0\) such that \(ab=0\) or \(ba=0\text{.}\) Multiplying either of these equations by \(a^{-1}\) gives \(b=0\text{,}\) a contradiction.

    Problem 1.16.

    Give an example of a ring \(R\) and an element \(a\in R\) which is neither a unit nor a zero-divisor.

    Problem 1.17.

    Prove that if \(R\) is finite then every element is either a unit of a zero-divisor.

    Definition 1.18.

    An element \(a\) of a ring \(R\) is called nilpotent if \(a^n = 0\) for some integer \(n \geq 1\text{.}\)

    Lemma 1.19.

    If \(a\) is a nilpotent element in a unital ring \(R\text{,}\) then \(1-a\) is a unit.

    Problem 1.20.

    Let \(x\) be a nilpotent element] of a ring. Show that \(1 + x\) is a unit of \(A.\) Deduce that the sum of a nilpotent element and a unit is a unit.

    Definition 1.21.

    Let \(R\) be a ring with identity. An element \(e\in R\) is called idempotent if \(e^2=e\text{.}\)

    Problem 1.22.

    Let \(R\) be a ring with identity.
    1. Give an example of a ring which has an idempotent other than \(0\) or \(1\text{.}\)
    2. Prove that if \(e\) is idempotent, so is \(1-e\text{.}\)
    3. Suppose \(R\) is commutative and \(e\) an idempotent. Let \(Re:=\{re\mid r\in R\}\text{.}\) Prove that \(Re\) is a commutative ring (with identity \(e\)).

    Problem 1.23.

    Let \(\Z[i] = \{a + bi \mid a,b \in \Z\}\) be the ring of Gaussian integers. Define a function
    \begin{equation*} N:\Z[i]\to\Z, N(a+bi)=a^2+b^2. \end{equation*}
    1. Prove that \(N(\a\b)=N(\a)N(\b)\) for any \(\a,\b\in \Z[i]\text{.}\)
    2. Use the function \(N\) to determine the units of \(\Z[i]\text{.}\)
    Proof.
    1. Let \(\alpha,\beta\in\Z[i]\) and behold \(\begin{align}N(\alpha\beta)&=N((a+bi)(c+di))\\&=N(ac-bd+adi+bci)\\&=(ac-bd)^2+(ad+bc)^2\\&=((ac)^2-2acbd+(bd)^2)+((ad)^2+2abcd+(bc^2))\\&=a^2c^2+b^2d^2+a^2d^2+b^2c^2\\&=(a^2c^2+a^2d^2)+(b^2c^2+b^2d^2)\\&=a^2(c^2+d^2)+b^2(c^2+d^2)\\N(\alpha)N(\beta)&=(a^2+b^2)(c^2+d^2).\end{align}\) So there's that done.
    2. Suppose \(\alpha\beta=1\text{.}\) Then by Part (a) we know
      \begin{equation*} N(\alpha)N(\beta)=N(\alpha\beta)=N(1)=1, \end{equation*}
      so \((a^2+b^2)(c^2+d^2)=1\text{,}\) with \(a,b,c,d\in\Z\text{.}\) Thus we have \(a^2+b^2=1=c^2+d^2\text{.}\) So the units of \(\Z[i]\) are \(\{\pm 1,\pm i\}\text{.}\)

    Problem 1.24.

    Let \(S=\{a+bi+cj+dk \mid a,b,c,d \in \Z\}\) be the ring of integral Hamiltonian quaternions, where multiplication is defined using the same rules as in \(\mathbb{H}\text{.}\) Define a function
    \begin{equation*} N:S\to\Z, N(a+bi+cj+dk)=a^2+b^2+c^2+d^2. \end{equation*}
    1. For any \(\a=a+bi+cj+dk\in S\text{,}\) denote \(\bar\a=a-bi-cj-dk\text{.}\) Show that \(\overline{\a\b}=\bar \b \bar\a\) for all \(\a,\b\in \mathbb{H}\text{.}\)
    2. Prove that \(N(\a)=\a\bar\a\) and \(N(\a\b)=N(\a)N(\b)\) for any \(\a,\b\in S\text{.}\)
    3. Prove that \(\a\in S^\times\) if and only if \(N(\a)=1\text{.}\)
    4. Show that there is a group isomorphism \((S^\times,\cdot)\cong Q_8\text{.}\)
