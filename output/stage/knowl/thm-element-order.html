Theorem 1.26. Element Order.

Let \(G\) be a group and \(x \in G\) any element.
  1. If \(|x| = n\text{,}\) then \(e, x, \dots, x^{n-1}\) are all distinct elements of \(G\text{.}\)
  2. If \(|x| = \infty\text{,}\) then \(x^i \ne x^j\) for all integers (positive or negative) \(i,j\) with \(i \ne j\text{.}\)
  3. \(|x| \leq |G|\text{.}\)
  4. If \(x^m = e\) then \(|x| \mid m\text{.}\)
