Item 4.
Let \(E=L^{N}\) for a normal subgroup \(N\) of \(G\text{.}\) We want to show that \(\operatorname{Gal}(E / F)\) is isomorphic to \(G / N\text{.}\)
For each \(\sigma \in G\text{,}\) we claim that \(\sigma(E) \subseteq E\text{.}\) By Lemma 6.10, for all \(\alpha \in E\) the element \(\sigma(\alpha)\) is also a root of \(m_{\alpha, F}\text{.}\) But since \(E / F\) is Galois, it must contain all of the roots of \(m_{\alpha, F}\text{,}\) by Corollary 6.26, so \(\sigma(\alpha) \in E\text{.}\) Thus \(\sigma(E) \subseteq E\text{,}\) so the restriction of \(\sigma\) to \(E\) determines an injective field homomorphism \(\left.\sigma\right|_{E}: E \rightarrow E\text{.}\) Since \(\left.\sigma\right|_{F}=\mathrm{id}_{F}\text{,}\) this map is also a linear transformation of vector spaces over \(F\text{.}\) But \(E\) is a finite vector space over \(F\text{,}\) and any injective linear transformation \(E \rightarrow E\) must be bijective. We conclude that \(\left.\sigma\right|_{E}\) is an automorphism of \(E\text{.}\) We thus have a well-defined function
\begin{equation*} \begin{gathered} \phi: G \longrightarrow \operatorname{Gal}(E / F) \\ \sigma \longmapsto \phi(\sigma)=\left.\sigma\right|_{E} . \end{gathered} \end{equation*}
Moreover, this map is a group homomorphism by construction. The kernel is the subgroup of \(G\) of automorphisms that restrict to the identity on \(E\text{,}\) which is precisely \(N\text{.}\) Hence we have an induced injective group homomorphism
\begin{equation*} \bar{\phi}: G / N \rightarrow \operatorname{Gal}(E / F) . \end{equation*}
But \(|N|=|\operatorname{Gal}(E / F)|<\infty\text{,}\) so this map \(\bar{\phi}\) must be an isomorphism.
