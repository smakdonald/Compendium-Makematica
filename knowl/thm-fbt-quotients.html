Theorem A.32. FBT = Function Building Theorem for Quotient Sets.

Let \sim be an equivalence relation on a set X, and let f: X \to Y be a function satisfying the property that whenever x,x' \in X and x \sim x' then f(x) = f(x'). Then:
  1. There is a well-defined function g:X/\sim \to Y defined by g([x]) = f(x) for all [x] in X/\sim ; that is, g \circ q = f, where q is the equivalence map.
  2. If f is onto, then g is onto.
  3. If f also satisfies the property that whenever x,x' \in X and f(x) = f(x') then x \sim x', then g is one-to-one.
